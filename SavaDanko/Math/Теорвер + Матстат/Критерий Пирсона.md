Критерий согласия Пирсона ($\chi^2$) — это статистический метод, который используется для проверки того, насколько наблюдаемое распределение частот согласуется с ожидаемым распределением, заданным гипотезой.

---
**⬩** Рассмотрим выборку из $n$ независимых наблюдений

**⬩** Сформулируем нулевую гипотезу $H_0$: выборка имеет распределение ... 
Гипотеза $H_0$ может быть уже задана: выборка следует некоторому теоретическому закону распределения (например, нормальному, экспоненциальному...)

**⬩** Разобьём область значений на $k$ дизъюнктных интервалов $I_1, I_2, \ldots, I_k$

**⬩** Посчитаем число наблюдаемых значений $O_i$ в интервале $I_i$

**⬩** Рассчитаем ожидаемые частоты $E_i = n \cdot p_i$, где $p_i$ — вероятность попасть в $I_i$ по предполагаемому распределению.

**⬩** Проверить применимость критерия: все $E_i \ge 5$.
Критерий Пирсона (хи-квадрат) работает корректно только при условии, что: $E_i \ge 5\quad \text{для всех интервалов } I_i$.
Это условие нужно, потому что формула $\chi^2 = \sum \frac{(O_i - E_i)^2}{E_i}$​ приближённо следует хи-квадрат-распределению только при больших $E_i$​ (по центральной предельной теореме).
Если $E_i$​ слишком мал (например, $< 5$), то распределение статистики становится смещённым и неточным, и вывод может быть недостоверным.

**⬩** Если $E_i < 5$
Объединить соседние интервалы, чтобы получить новый интервал с суммарной $E_i \ge 5$.  
Это называется объединением интервалов.

**⬩** Вычислить статистику $\chi^2$.
$\chi^2 = \sum_{i=1}^{k} \frac{(O_i - E_i)^2}{E_i}$

**⬩** Сравнить со значением $\chi^2_{k-1}$ из таблицы критических значений.

**⬩** Сделать вывод:    
    - Если $\chi^2_{\text{набл.}} < \chi^2_{\text{крит}}$, то $H_0$ не отвергается.
    - Иначе — отвергается на уровне значимости $\alpha$.

---
В критерии Пирсона ($\chi^2$-критерий согласия или независимости) символ обозначает значение тестовой статистики, вычисляемой по наблюдённым и ожидаемым частотам.
