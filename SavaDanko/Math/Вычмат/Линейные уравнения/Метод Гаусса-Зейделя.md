Метод Гаусса–Зейделя — это усовершенствованная версия метода простых итераций.

##### Основная идея:
- Метод заключается в последовательном обновлении значений неизвестных переменных, используя уже найденные значения на текущем шаге итерации.
- В отличие от метода Простых итераций, где все значения новой итерации рассчитываются одновременно, здесь каждое новое значение немедленно используется для вычисления следующих.
---
##### Пример:
Рассмотрим систему уравнений:
$\begin{cases} 4x_1 - x_2 + 0x_3 = 3 \\ -x_1 + 4x_2 - x_3 = 2 \\ 0x_1 - x_2 + 4x_3 = 1 \end{cases}$

##### Преобразуем к итерационной форме:
$\begin{cases} x_1 = \frac{3 + x_2}{4} \\ x_2 = \frac{2 + x_1 + x_3}{4} \\ x_3 = \frac{1 + x_2}{4} \end{cases}$

##### Итерации
##### Начальное приближение:
$x_1^{(0)} = 0$
$x_2^{(0)} = 0$
$x_3^{(0)} = 0$

##### 1-я итерация:
$x_1^{(1)} = \frac{3 + 0}{4} = 0{,}75$
$x_2^{(1)} = \frac{2 + 0{,}75 + 0}{4} = 0{,}6875$
$x_3^{(1)} = \frac{1 + 0{,}6875}{4} = 0{,}421875$

##### 2-я итерация:
$x_1^{(2)} = \frac{3 + 0{,}6875}{4} = 0{,}921875$
$x_2^{(2)} = \frac{2 + 0{,}921875 + 0{,}421875}{4} = 0{,}8359375$
$x_3^{(2)} = \frac{1 + 0{,}8359375}{4} = 0{,}458984375$

Продолжаем итерации до достижения необходимой точности $\varepsilon$. Метод быстро сходится, особенно если матрица системы обладает диагональным преобладанием.
