### Основная идея метода Адамса:
Метод Адамса основывается на **интерполяции производной** с помощью полиномов Ньютона. В зависимости от того, как вычисляется новая точка, методы Адамса делятся на:
1. **Явные методы Адамса-Бэшфорта** (предсказатели).
2. **Неявные методы Адамса-Мултона** (уточнители).

## Явный метод Адамса-Бэшфорта
Используется для **предсказания** нового значения.
#### Формула (2-шаговый метод):
$y_{n+1} = y_n + \frac{h}{2} [3f(x_n, y_n) - f(x_{n-1}, y_{n-1})]$

## Неявный метод Адамса-Мултона
Используется для **уточнения** значения после предсказания.
#### Формула (2-шаговый метод):
$y_{n+1} = y_n + \frac{h}{2} [f(x_{n+1}, y_{n+1}) + f(x_n, y_n)]$

### Схема работы метода Адамса
1. **Инициализация:**
    - Для первых точек (например, $x_0, x_1, x_2$) используем метод Рунге-Кутта.
2. **Предсказание:**
    - Вычисляем $y_{n+1}^*$ с помощью явного метода Адамса-Бэшфорта.
3. **Уточнение:**
    - Корректируем значение с помощью неявного метода Адамса-Мултона.
4. **Переход к следующей итерации.**

## Пример задачи:
Рассмотрим задачу Коши:
$\frac{dy}{dx} = y, \quad y(0) = 1$
Решим на интервале $[0, 0.5]$ с шагом $h = 0.1$

### Инициализация (Метод Рунге-Кутта для первых точек):
1. $x_0 = 0$, $y_0 = 1$
2. $x_1 = 0.1$, $y_1 = e^{0.1} \approx 1.1052$
3. $x_2 = 0.2$, $y_2 = e^{0.2} \approx 1.2214$

### Метод Адамса-Бэшфорта (2-го порядка):
$y_{n+1} = y_n + \frac{h}{2} [3f(x_n, y_n) - f(x_{n-1}, y_{n-1})]$

### Метод Адамса-Мултона (уточнение):
$y_{n+1} = y_n + \frac{h}{2} [f(x_{n+1}, y_{n+1}) + f(x_n, y_n)]$

- Корректируем предсказанное значение:
$y_{n+1} = y_n + \frac{h}{2} [f(x_{n+1}, y_{n+1}^*) + f(x_n, y_n)]$
