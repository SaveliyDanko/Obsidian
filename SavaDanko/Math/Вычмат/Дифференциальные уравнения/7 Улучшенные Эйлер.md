#### Постановка задачи Коши
Улучшенный метод Эйлера решает задачу Коши:
$\frac{dy}{dx} = f(x, y), \quad y(x_0) = y_0$
где:
- $\frac{dy}{dx}$ — производная функции;
- $f(x, y)$ — правая часть уравнения, заданная как функция;
- $x_0$ — начальная точка;
- $y_0$ — начальное значение функции в точке $x_0$.

### Идея метода
Метод Эйлера аппроксимирует значение функции на следующем шаге, используя **производную в начальной точке**.  
Улучшенный метод Эйлера делает два вычисления:
1. Находит **приближенное значение на следующем шаге** методом Эйлера.
2. Использует это значение для **уточнения производной на текущем шаге**.
3. Вычисляет среднее значение производной между начальной и предсказанной точкой.

### Формулы улучшенного метода Эйлера
1. **Шаг 1: Вычисляем промежуточное значение (предсказатель)**:
$k_1 = f(x_n, y_n)$
$\tilde{y}_{n+1} = y_n + h \cdot k_1$

2. **Шаг 2: Корректируем значение (уточнитель)**:
$k_2 = f(x_n + h, \tilde{y}_{n+1})$
$y_{n+1} = y_n + \frac{h}{2} \cdot (k_1 + k_2)$
