**Задача Коши** — это задача нахождения решения обыкновенного дифференциального уравнения (ОДУ) при заданном начальном условии.

### Общий вид задачи Коши для ОДУ первого порядка:
Пусть уравнение имеет вид:
$\frac{dy}{dx} = f(x, y)$
где:
- $x$ — независимая переменная;
- $y$ — неизвестная функция от xx;
- $f(x, y)$ — заданная функция двух переменных.

**Начальное условие:**
$y(x_0) = y_0$
где:
- $x_0$ — начальная точка (значение независимой переменной);
- $y_0$ — значение искомой функции в этой точке.

### Постановка задачи Коши:
Найти функцию $y = \varphi(x)$, которая:
1. Удовлетворяет дифференциальному уравнению:
$\frac{dy}{dx} = f(x, y)$
2. Соответствует начальному условию:
$\varphi(x_0) = y_0$

### Пример задачи Коши:
Рассмотрим конкретное уравнение:
$\frac{dy}{dx} = y, \quad y(0) = 1$

**Интерпретация:**
- Уравнение: $y' = y$ — это уравнение экспоненциального роста.
- Начальное условие: при $x = 0$, $y = 1$.

**Решение:**  
Общее решение данного уравнения:
$y = Ce^x$

Подставим начальное условие:
$1 = Ce^0 \implies C = 1$

Тогда конкретное решение задачи Коши:
$y = e^x$
