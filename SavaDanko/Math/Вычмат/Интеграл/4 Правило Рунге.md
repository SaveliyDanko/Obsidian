Это правило, которое позволяет оценить погрешность численного метода, если мы знаем результат вычисления с двумя разными шагами.

#### Общая идея:
Если вы применили один и тот же метод (например, трапеций или Симпсона) на одном и том же отрезке $[a,b]$ с:
- шагом $h$   
- и с шагом $2h/2$
и получили результаты:
- $I_h$ — приближённое значение с шагом $h$    
- $I_{h/2}$ — приближённое значение с шагом $h/2$
то оценка погрешности $R$ и улучшенное значение интеграла находятся по формулам:

#### Формула Рунге для погрешности:
$$R = \frac{I_{h/2} - I_h}{2^K - 1}$$
где:
- $k$ — порядок точности метода
- $R$ — приближённая погрешность, которую допустил метод при шаге $h/2$

#### Значения $k$ для популярных методов:

| Метод           | Порядок $k$ |
| --------------- | ----------- |
| Прямоугольников | 2           |
| Трапеций        | 2           |
| Симпсона        | 4           |

#### Улучшенное значение интеграла (с повышенной точностью):
$$I \approx I_{h/2} + R = I_{h/2} + \frac{I_{h/2} - I_h}{2^p - 1}$$
