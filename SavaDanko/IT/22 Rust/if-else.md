**if-else**
```rust
fn main() {
    if condition == 0 {
        // some code
    } else if number % 3 == 0 {
        // some code
    } else {
        // some code
    }
}
```

**Using if in a let Statement**
```rust
fn main() {
    let condition = true;
    let number = if condition { 5 } else { 6 };
}
```
