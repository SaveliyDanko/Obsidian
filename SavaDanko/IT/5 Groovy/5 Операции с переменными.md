```groovy
1  + 2 == 3
4  - 3 == 1
3  * 5 == 15
3  / 2 == 1.5
10  % 3 == 1
2 ** 3 == 8
```
Отличия с Java:
- появилась операция возведения в степенть `**`
- по умолчанию деление происходит в вещественном виде `3  / 2 == 1.5`, чтобы произвести целочисленное деление нужно вызвать `Integer.intdiv()`
- Существует возможность продвинутого приведения типов с помощью оператора `as`

Появление `identical` и `not identical` операторов, сравнение не по значению, а по ссылке: 
```groovy
===
!==
```

Операторы в Groovy можно применять к любым класса, у которых реализован метод `Comparable()`
