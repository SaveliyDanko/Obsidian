Groovy — это динамический язык программирования, работающий поверх JVM (Java Virtual Machine).

Существует возможность создавать Groovy Script

Необязательное использование `;` и `()` после методов

Функционал Groovy под капотом содержит Java

Вместо `System.out.println()` используем `println`

В Groovy по умолчанию используется модификатор доступа public
`package-private` модификатор доступа устанавливается с помощью аннотации `@PackageScope`

В Groovy по умолчанию для полей(properties) используется модификатор доступа private

Примитивные типы данных представлены обертками

Динамическое определение типов с помощью def (аналог var в Java).
Можно применять к переменным, полям, возвратам их методов
