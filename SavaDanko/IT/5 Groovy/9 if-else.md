###### **Отсутствие обязательного объявления типа**
```groovy
def name = "Groovy"

if (name) {
    println "Строка не пуста"
}
```
В отличие от строгой Java, Groovy использует концепцию truthy/falsy.

###### **Что считается true и false в Groovy**

| Значение                         | Оценивается как |
| -------------------------------- | --------------- |
| `null`                           | `false`         |
| Пустая строка `""`               | `false`         |
| Пустой список `[]` или map `[:]` | `false`         |
| Число `0` или `0.0`              | `false`         |
| Все остальные значения           | `true`          |


###### **Элвис-оператор `?:`**
Специальная groovy-расширенная версия тернарного оператора:
```groovy
def name = null
def displayName = name ?: "Аноним"

println displayName
```
Если `name` == `null` или falsy, будет использовано значение по умолчанию `"Аноним"`.
