#bash 

---
```bash
name="Bob"

echo "$name"
# Bob
```

Bash не различает типы данных (всё — строки), но можно имитировать:
```bash
number1=10
number2=5
sum=$((number1 + number2))  # Арифметика
```

| Переменная | Значение                     |
| ---------- | ---------------------------- |
| `$0`       | Имя скрипта                  |
| `$1`, `$2` | Аргументы командной строки   |
| `$@`       | Все аргументы (как массив)   |
| `$#`       | Кол-во аргументов            |
| `$$`       | PID текущего процесса        |
| `$?`       | Код выхода последней команды |

Удаление переменной
```bash
unset name
```
