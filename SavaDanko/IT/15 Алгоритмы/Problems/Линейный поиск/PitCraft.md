**Условие:**
Игра **PitCraft** происходит в двумерном мире, состоящем из блоков размером 1 на 1 метр.  
Остров игрока — это набор столбцов различной высоты, состоящих из каменных блоков и окружённых морем.
Над островом прошёл сильный дождь, который заполнил водой все низины. Вся лишняя вода, не помещающаяся в этих низинах, стекла в море (уровень моря не поднялся).

**Задача:**  
По заданному ландшафту острова определить, сколько единичных блоков воды осталось после дождя в низинах.

#### ✅ Решение:
```java
public class WaterTrap {
    public static void main(String[] args) {
        // Входной массив: высоты столбцов земли
        int[] arr = {3, 1, 1, 1, 3, 4, 5, 1, 1, 3};

        int sum = 0; // Общее количество воды
        int maxItem = arr[0]; // Максимальная высота
        int maxItemInd = 0;   // Индекс самой высокой точки

        // Шаг 1: Находим индекс самого высокого столбца
        for (int i = 0; i < arr.length; i++) {
            if (arr[i] > maxItem) {
                maxItem = arr[i];
                maxItemInd = i;
            }
        }

        // Шаг 2: Слева до самого высокого столбца
        int curItemLeft = arr[0];
        for (int i = 0; i < maxItemInd; i++) {
            if (arr[i] < curItemLeft) {
                sum += curItemLeft - arr[i]; // Добавляем объем воды, который может быть на этом месте
            } else {
                curItemLeft = arr[i]; // Обновляем текущую левую границу
            }
        }

        // Шаг 3: Справа до самого высокого столбца
        int curItemRight = arr[arr.length - 1];
        for (int i = arr.length - 1; i > maxItemInd; i--) {
            if (arr[i] < curItemRight) {
                sum += curItemRight - arr[i]; // Добавляем объем воды с правой стороны
            } else {
                curItemRight = arr[i]; // Обновляем текущую правую границу
            }
        }

        // Вывод результата
        System.out.println("Объём воды после дождя: " + sum + " блоков");
    }
}
```
