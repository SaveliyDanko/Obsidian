Списки в HTML используются для группировки связанных элементов. HTML поддерживает три типа списков: упорядоченные, неупорядоченные и списки определений.

**Упорядоченные списки** (`<ol>`)
```html
<ol>
	<li>Первый элемент</li>
	<li>Второй элемент</li>
	<li>Третий элемент</li>
</ol>
```
С помощью атрибута `type` можно изменить стиль нумерации.

**Неупорядоченные списки** (`<ul>`)
Используется, когда порядок элементов не имеет значения. Каждый пункт отображается с маркером (точкой, кружком, квадратом...)
```html
<ul>
	<li>Первый элемент</li>
	<li>Второй элемент</li>
	<li>Третий элемент</li>
</ul>
```

**Список определений** (`<dl>`)
```html
<dl>
	<dt>Термин 1</dt>
	<dd>Определение термина 1</dd>

	<dt>Термин 2</dt>
	<dd>Определение термина 2</dd>
</dl>
```

Можно создавать вложенные списки:
```html
<ul>
	<li>Элемент 1
		<ul>
			<li>Подэлемент 1.1</li>
			<li>Подэлемент 1.2</li>
		</ul>
	</li>

	<li>Элемент 2</li>
</ul>
```
