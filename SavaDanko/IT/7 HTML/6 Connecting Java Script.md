```html
<script src="script.js"></script>
```
Можно подключить несколько скриптов, используя несколько тегов `<script>`

Располагаем в конце секции `<body>`, скрипт загружается после контента, улучшая производительность
```html
<body>
	<!-- Контент страницы -->
	<script src="script.js"></script>
</body>
```
