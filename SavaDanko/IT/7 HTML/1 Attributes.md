HTML-атрибуты предоставляют дополнительную информацию об элементах HTML.
==1. Базовый синтаксис:==
```html
<a name="value">Link</a>
```

==2. Наиболее часто используемые атрибуты:==
- `class` - используется для CSS-стилизации
- `id` - уникальный идентификатор элемента
- `src` - URL источника
- `alt` - альтернативный текст для изображения
- `href` - URL связанного ресурса
- `style` - встроенные стили CSS
- `title` - дополнительная информация об элементе
