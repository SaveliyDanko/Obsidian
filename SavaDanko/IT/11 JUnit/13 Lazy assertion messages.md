Lazy assertion messages — это механизм, который позволяет откладывать формирование текста сообщения об ошибке до того момента, пока ошибка действительно произойдёт.
Иначе говоря, сообщение создаётся только если assert упал, а не заранее.

###### **Зачем нужны lazy-messages?**
Обычные (не ленивые) сообщения формируют строку всегда, даже если тест проходит. Это может быть дорого по вычислениям.

Lazy-messages позволяют писать:
```java
assertEquals(expected, actual, () -> createVeryLongMessage());
```
Функция `createVeryLongMessage()` будет вызвана только если сравнение не прошло.
