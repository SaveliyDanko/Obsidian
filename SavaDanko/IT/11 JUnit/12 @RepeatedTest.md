Аннотация, которая позволяет запускать один и тот же тест несколько раз подряд автоматически. Она особенно полезна, когда нужно проверить стабильность и надёжность логики.

Аннотация заменяет `@Test` и принимает количество повторений:
```java
@RepeatedTest(5)
void testShouldRunFiveTimes() {
    System.out.println("Выполнение теста");
}
```
Этот тест выполнится пять раз, и каждый запуск будет считаться отдельным прогоном в отчёте.
