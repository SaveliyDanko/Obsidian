Аннотация, которая позволяет отключить тест или весь класс тестов, временно исключив их из запуска.

Она работает как комментарий с пояснением, но правильным, контролируемым способом, и тест остаётся виден в отчётах.
```java
@Disabled("Падает из-за ошибки в сервисе, исправлю позже")
@Test
void shouldValidateUser() {
    // тест не будет выполнен
}
```
JUnit покажет этот тест в отчёте как skipped.
