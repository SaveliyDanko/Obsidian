#web

---
**Gulp** — это инструмент для автоматизации задач (task runner) в веб-разработке. Он помогает упростить и ускорить повторяющиеся процессы, такие как:
- минификация CSS/JS
- компиляция препроцессоров (SASS, LESS и др.)
- объединение файлов (concat)
- оптимизация изображений
- запуск локального сервера и автообновление страницы (LiveReload), 
- слежение за изменениями в файлах (watch)
    
Gulp работает с помощью **stream'ов (потоков)** в Node.js. Это значит, что файлы читаются, обрабатываются и сохраняются в цепочке, не записываясь на диск между шагами. Это делает его **очень быстрым** по сравнению с другими сборщиками.

#### Как работает Gulp?
1. Устанавливается Node.js и npm — Gulp работает как npm-пакет.
2. Устанавливается Gulp глобально и в проект:
    ```bash
    npm install --global gulp-cli
    npm install --save-dev gulp
    ```
3. Создаётся файл `gulpfile.js`, в котором описываются задачи.
4. Запускается команда `gulp имя_задачи`.
