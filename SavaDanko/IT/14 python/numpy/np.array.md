`np.array` — это _n-мерный массив_.

```python
import numpy as np
```

##### Внутренние свойства `array`
```python
a = np.array([[1, 2, 3], [4, 5, 6]])

print(a.shape)     # (2, 3) — форма (2 строки, 3 столбца)
print(a.ndim)      # 2 — количество измерений
print(a.size)      # 6 — общее число элементов
print(a.dtype)     # int64 — тип элементов
print(a.itemsize)  # 8 — байт на элемент
```

##### Как создать `np.array`
**1. Из обычного списка:**
```python
a = np.array([1, 2, 3])  # 1D
b = np.array([[1, 2], [3, 4]])  # 2D
```

**2. Указание типа данных:**
```python
a = np.array([1, 2, 3], dtype=float)
```

**3. Из других источников:**
```python
np.zeros((2, 3))          # массив из нулей
np.ones((3, 3))           # массив из единиц
np.full((2, 2), fill_value=7)  # массив из семёрок
np.eye(3)                 # единичная матрица
np.arange(0, 10, 2)       # [0, 2, 4, 6, 8]
np.linspace(0, 1, 5)      # 5 чисел от 0 до 1
```

##### Операции с `np.array`
NumPy работает векторно — то есть над массивами сразу:
```python
a = np.array([1, 2, 3])
b = np.array([10, 20, 30])

a + b      # [11 22 33]
a * b      # [10 40 90]
a ** 2     # [1 4 9]
np.sin(a)  # [sin(1), sin(2), sin(3)]
```
Операции происходят поэлементно.
