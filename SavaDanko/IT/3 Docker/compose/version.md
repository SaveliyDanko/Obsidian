Параметр `version` указывает формат (схему) файла `docker-compose.yml`, то есть, какую версию синтаксиса Docker Compose использует ваш конфигурационный файл.

Начиная с Docker Compose 1.27.0, формат `version` в явном виде устарел и стал необязательным, поскольку Docker перешёл к единому Compose Specification.
