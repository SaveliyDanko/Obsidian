Параметр `attachable: true` определяет, можно ли вручную подключать контейнеры к этой сети, если она уже создана.

Когда сеть помечена как attachable:
- К ней можно вручную присоединять контейнеры через команду `docker network connect`.
- Это удобно для отладки, для временных контейнеров или для сторонних сервисов, которые не описаны в `docker-compose.yml`.
