**Docker Container** — это работающее приложение, созданное на основе Docker Image (шаблона).

Контейнер:
- Изолирован (как мини-компьютер внутри твоего)
- Содержит всё, что нужно для запуска (код, библиотеки, окружение)
- Быстро запускается, легко удаляется и переносится

#### Жизненный цикл контейнера (Lifecycle
Контейнер проходит несколько состояний:
```
[СОЗДАН] → [ЗАПУЩЕН] → [ОСТАНОВЛЕН] → [УДАЛЁН]
```
