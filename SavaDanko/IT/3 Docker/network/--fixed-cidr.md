Флаг `--fixed-cidr` задаёт фиксированный диапазон IP-адресов для назначения контейнерам внутри мостовой сети Docker (`docker0`).

Проще говоря:
- `--bip` задаёт IP-подсеть для интерфейса `docker0`.
- `--fixed-cidr` ограничивает диапазон IP-адресов, из которых Docker может назначать адреса контейнерам.
