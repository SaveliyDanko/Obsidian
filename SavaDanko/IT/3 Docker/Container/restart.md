`docker restart` используется для перезапуска контейнера.
- Контейнер получает сигнал `SIGTERM`, чтобы корректно завершить процессы.
- Через 10 секунд по умолчанию (или меньше, если задан флаг), если не завершился — отправляется `SIGKILL`.
- Затем контейнер запускается заново, как при `docker start`.
```zsh
docker restart [OPTIONS] CONTAINER [CONTAINER...]
```
