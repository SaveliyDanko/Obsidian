
---
**`<containerPort>`**  
Контейнерный порт пробрасывается на случайный (динамический) порт хоста.
```bash
docker run -p 8080 myapp
```
- Порт `8080` внутри контейнера становится доступен снаружи.
- Снаружи этот порт будет автоматически проброшен на случайный свободный порт хоста.
```
0.0.0.0:49154->8080/tcp
```
    
---
**`<hostPort>:<containerPort>`**  
Порт контейнера явно пробрасывается на указанный порт хоста на всех интерфейсах.
```bash
docker run -p 8080:80 nginx
```

---
**`<ip>::<containerPort>`**
Порт контейнера пробрасывается на случайный порт хоста, но только для указанного IP-интерфейса.
```bash
docker run -p 192.168.0.32::8080 myapp
```

---
**`<ip>:<hostPort>:<containerPort>`**  
Контейнерный порт явно пробрасывается на указанный порт хоста, и это проброс доступен только с указанного IP-интерфейса.
```bash
docker run -p 192.168.0.32:8080:80 nginx
```
