**Сущность (Entity)** — это объект из реального мира, о котором мы хотим хранить информацию в базе данных.

Примеры сущностей:
- Пользователь (`User`)    
- Продукт (`Product`)
- Заказ (`Order`)
- Студент (`Student`)
- Книга (`Book`)
    
###### **В таблицах:**
Обычно сущность = таблица в базе данных.
Например, сущность `User` может быть представлена как таблица `users` с данными о каждом пользователе.
