**DDL (Data Definition Language)** — это процесс создания, изменения и удаления структуры базы данных: таблиц, типов, связей, индексов, представлений и т.д.

###### **Что входит в определение данных:**

| Операция   | Пример SQL                                            | Назначение                                               |
| ---------- | ----------------------------------------------------- | -------------------------------------------------------- |
| `CREATE`   | `CREATE TABLE users (...);`                           | Создание таблиц, индексов, представлений, типов          |
| `ALTER`    | `ALTER TABLE users ADD COLUMN email TEXT;`            | Изменение структуры таблицы или типа                     |
| `DROP`     | `DROP TABLE users;`                                   | Удаление объекта БД                                      |
| `TRUNCATE` | `TRUNCATE TABLE logs;`                                | Быстрое удаление всех данных из таблицы без логгирования |
| `COMMENT`  | `COMMENT ON COLUMN users.name IS 'Имя пользователя';` | Добавление описания                                      |
