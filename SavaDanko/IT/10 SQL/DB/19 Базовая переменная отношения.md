###### **Базовые переменные отношения (Base Relation Variables)**
Базовые переменные отношения (базовые отношения, base relation, base table) — это таблицы, которые физически хранят данные в базе. Они являются основой всей структуры хранения.
```sql
CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    name TEXT,
    age INT
);
```
Здесь `users` — это базовая переменная отношения. Она связана с файлом или страницами в диске, где физически хранятся строки (кортежи).
