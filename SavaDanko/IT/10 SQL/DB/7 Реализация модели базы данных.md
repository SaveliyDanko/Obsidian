Когда ты описал, какие данные тебе нужны — наступает момент, когда это нужно реализовать в коде или базе данных. Это и есть реализация модели данных.

Создаются таблицы, столбцы, связи и ограничения.

Пример реализации таблицы `User` в SQL:
```sql
CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100) UNIQUE,
    password VARCHAR(100)
);
```


###### **Связь между таблицами (отношения):**
#### SQL:
```sql
CREATE TABLE orders (
    id SERIAL PRIMARY KEY,
    user_id INT REFERENCES users(id),
    order_date TIMESTAMP
);
```
