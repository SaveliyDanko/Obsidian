**Отношение** — это связь между двумя (или более) сущностями.

Сущности редко существуют изолированно. Пользователи делают заказы, книги находятся у авторов и т.д.

###### **Примеры отношений:**
1. User → Order
    Один пользователь может сделать много заказов.
    
2. Book → Author
    У одной книги может быть один автор.
    
3. Student ↔ Course
    Студент может посещать много курсов, и каждый курс может быть у многих студентов (многие ко многим).
    

---

###### **Виды отношений:**

| Тип                    | Описание                                     | Пример                 |
| ---------------------- | -------------------------------------------- | ---------------------- |
| Один к одному (1:1)    | У каждой сущности есть только одна связанная | Пользователь ↔ Паспорт |
| Один ко многим (1:N)   | Один связан с несколькими                    | Клиент → Заказы        |
| Многие ко многим (N:M) | Несколько к нескольким                       | Студенты ↔ Курсы       |
