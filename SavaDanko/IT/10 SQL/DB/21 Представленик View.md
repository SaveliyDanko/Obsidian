###### **Представление (View)**
Представление (`VIEW`) — это логическая, виртуальная таблица, которая не хранит данные, а строится на основе SQL-запроса.
```sql
CREATE VIEW adult_users AS
SELECT id, name FROM users WHERE age >= 18;
```
- `adult_users` — это представление    
- Это производная переменная отношения
- Когда ты делаешь `SELECT * FROM adult_users`, СУБД выполняет вложенный SQL, как подзапрос
