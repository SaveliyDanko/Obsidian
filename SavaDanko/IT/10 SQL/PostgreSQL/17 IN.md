Оператор `IN` в PostgreSQL используется для проверки принадлежности значения множеству. Это удобная и читаемая форма записи нескольких условий через `OR`.

###### **Синтаксис**
```sql
SELECT ...
FROM ...
WHERE выражение IN (значение1, значение2, ...);
```

###### **Простое использование**
```sql
SELECT * FROM users
WHERE country IN ('Germany', 'France', 'Italy');
```
Найдёт всех пользователей из указанных стран.

###### **С подзапросом**
```sql
SELECT * FROM users
WHERE id IN (
  SELECT user_id FROM orders WHERE total > 500
);
```
Вернёт всех пользователей, у которых есть заказ на сумму более 500.

###### **С числовыми значениями**
```sql
SELECT * FROM products
WHERE category_id IN (1, 2, 3);
```

###### **Проверка на кортежи (row values)**
```sql
SELECT * FROM logs
WHERE (level, source) IN (('ERROR', 'backend'), ('WARN', 'frontend'));
```
Проверяет комбинацию значений в двух столбцах.

###### **`NOT IN`**
Противоположность `IN`:
```sql
SELECT * FROM users
WHERE country NOT IN ('Russia', 'Belarus');
```
