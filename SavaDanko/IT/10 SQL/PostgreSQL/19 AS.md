В PostgreSQL ключевое слово `AS` используется для переименования (алиасов) столбцов, таблиц, подзапросов, выражений и других объектов в SQL-запросах.

###### **Алиас (псевдоним) для столбца**
```sql
SELECT name AS full_name, age AS years
FROM users;
```
Столбцы в результате будут названы `full_name` и `years`.

Аналог без `AS`:
```sql
SELECT name full_name, age years FROM users;
```
Работает — но лучше использовать `AS` для читаемости.

###### **Алиас для таблицы**
```sql
SELECT u.name, o.total
FROM users AS u
JOIN orders AS o ON u.id = o.user_id;
```
`u` и `o` — короткие имена для таблиц. Это удобно в `JOIN`, `WHERE`, `ON`, особенно при длинных именах.

###### **Алиас для выражения**
```sql
SELECT price * quantity AS total_price
FROM order_items;
```
Даёт понятное имя вычисленному значению.

###### **Алиас для подзапроса**
```sql
SELECT sub.user_id, sub.order_count
FROM (
  SELECT user_id, COUNT(*) AS order_count
  FROM orders
  GROUP BY user_id
) AS sub;
```
Обязателен: подзапрос должен быть именован.
