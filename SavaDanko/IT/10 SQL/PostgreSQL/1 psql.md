Команда `psql` — это официальный интерактивный терминал PostgreSQL. 

```bash
psql [OPTIONS] [DBNAME [USERNAME]]
```
Если база данных и имя пользователя не указаны, `psql` попытается использовать имя текущего пользователя Linux.

###### Основные ключи `psql`

| Ключ            | Назначение                                                                     |
| --------------- | ------------------------------------------------------------------------------ |
| `-h <host>`     | Указать хост (например, `localhost`, `127.0.0.1`, IP, имя контейнера в Docker) |
| `-p <port>`     | Указать порт (по умолчанию: `5432`)                                            |
| `-U <username>` | Пользователь для подключения                                                   |
| `-d <dbname>`   | Имя базы данных                                                                |
| `-f <file.sql>` | Выполнить SQL-скрипт из файла                                                  |
| `-c "<SQL>"`    | Выполнить одну SQL-команду и выйти                                             |
| `-W`            | Принудительно запросить пароль                                                 |
| `--help`        | Справка по ключам                                                              |

###### Внутренние команды в интерактивном режиме `psql`
Внутри `psql` можно использовать не только SQL, но и служебные команды PostgreSQL, начинающиеся с `\`.

| Команда      | Назначение                    |
| ------------ | ----------------------------- |
| `\l`         | Список баз данных             |
| `\c <db>`    | Подключиться к другой базе    |
| `\dt`        | Список таблиц                 |
| `\d <table>` | Описание таблицы              |
| `\du`        | Список пользователей          |
| `\q`         | Выход                         |
| `\?`         | Список всех внутренних команд |
