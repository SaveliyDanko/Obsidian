###### **`LIKE` — чувствителен к регистру**
Сравнивает строку с шаблоном, где:
- `%` — соответствует любой последовательности символов (в том числе пустой)
- `_` — соответствует одному символу
    
###### Пример:
```sql
SELECT * FROM users
WHERE name LIKE 'A%';
```

###### **`ILIKE` — нечувствителен к регистру**
```sql
SELECT * FROM users
WHERE name ILIKE 'a%';
```
