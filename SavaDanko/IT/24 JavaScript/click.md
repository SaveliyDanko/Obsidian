**`click`** — это событие, которое срабатывает, когда пользователь кликает на элемент левой кнопкой мыши (или сенсорным эквивалентом на мобильных устройствах), либо когда элемент активируется с клавиатуры (например, кнопка через `Enter` или `Space`).

###### **Синтаксис**
```js
element.addEventListener('click', handler);
```
- `element` — любой элемент DOM (HTML-узел, `document`, `window`).
- `handler` — функция, которая примет объект события `MouseEvent`.
    
###### **Объект события (`MouseEvent`)**
```js
element.addEventListener('click', e => {
  console.log(e.type);      // 'click'
  console.log(e.clientX);   // координата X клика относительно окна
  console.log(e.clientY);   // координата Y клика
  console.log(e.target);    // элемент, на котором произошло событие
});
```

###### **Особенности**
1. Всплытие — `click` всплывает до `document`, что позволяет использовать делегирование событий.
2. Порядок — перед `click` срабатывают:
    - `mousedown`
    - `mouseup` 
3. Программный вызов
    ```js
    element.click(); // вызовет событие click
    ```
    При этом `MouseEvent` создаётся, но координаты будут `0,0`.
