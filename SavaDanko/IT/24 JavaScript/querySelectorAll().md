`querySelectorAll()` — это метод объекта `Document` или любого DOM-элемента, который возвращает все элементы, соответствующие указанному CSS-селектору, в виде неживого (`static`) объекта `NodeList`.  
Если ничего не найдено, возвращает пустой `NodeList`.

###### **Синтаксис**
```js
const elements = parent.querySelectorAll(selectors);
```
- `parent` — `document` или элемент, внутри которого идёт поиск.
- `selectors` — строка с валидным CSS-селектором (можно комбинировать теги, классы, id, псевдоклассы).

###### **Возвращаемое значение**
- `NodeList` — объект, похожий на массив, но:
    - Можно обращаться по индексу: `elements[0]`.
    - Есть свойство `.length`.
    - Можно использовать `for...of` и `forEach`.
    - Нет всех методов массива (`map`, `filter`, `reduce`), если не преобразовать в массив:
        ```js
        const arr = Array.from(elements);
        ```
