The WheelEvent Object handles events that occur when the mouse wheel is scrolling.

**Свойства `WheelEvent`**
Вот свойства WheelEvent из твоего изображения в текстовом виде:
- `deltaX` — величина горизонтальной (по оси X) прокрутки колеса мыши.
- `deltaY` — величина вертикальной (по оси Y) прокрутки колеса мыши.
- `deltaZ` — величина прокрутки по оси Z (редко используется, в основном для 3D-взаимодействий).
- `deltaMode` — единицы измерения прокрутки: пиксели, строки или страницы (`pixels`, `lines`, `pages`).
