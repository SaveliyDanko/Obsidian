`"use strict"` – это директива в JavaScript, которая переводит движок JavaScript в **строгий режим**.

###### **Что меняет "use strict":**
1. Запрет неявных объявлений: 
- В обычном режиме, если присвоить значение необъявленной переменной, JavaScript создаст глобальную переменную.
- В строгом режиме это вызовет ошибку.
```javascript
"use strict";
x = 10; // Ошибка: x is not defined
```

2. Запрет на дублирование параметров функций:  
- В строгом режиме нельзя объявлять функции с одинаковыми именами параметров.
```javascript
"use strict";
function myFunc(a, a) { } // Ошибка: Duplicate parameter name not allowed in this context
```

3. Запрет на использование зарезервированных слов:
- В будущем новые версии ECMAScript могут вводить новые ключевые слова. Использование этих слов в качестве идентификаторов (переменных, функций) в строгом режиме вызовет ошибку.

4. Более строгие правила для `this`:
- В строгом режиме `this` в функциях не ссылается на глобальный объект (`window` в браузерах) по умолчанию.
- Это помогает избежать непреднамеренных изменений глобальных объектов.

5. Запрет на `with`:
- Оператор `with` считается плохой практикой и запрещен в строгом режиме.

###### **Как использовать "use strict":**
- Глобально: Поместите `"use strict"` в начале скрипта.
```javascript
"use strict"; 
// весь код ниже будет выполняться в строгом режиме
```
  
- Локально (в функции): Поместите `"use strict"` в начале тела функции.
```javascript
function myFunc() {
"use strict"; 
// код внутри функции будет выполняться в строгом режиме
}
```
