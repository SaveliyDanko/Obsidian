`call()` — это встроенный метод всех функций в JavaScript, позволяющий явно задать `this` и передать аргументы по-одному при вызове.

###### **Базовый синтаксис**
```js
func.call(thisArg, arg1, arg2, ...);
```
- `thisArg` — значение, которое будет использовано как `this` внутри функции.
- `arg1, arg2, ...` — аргументы функции.
    
###### **Что делает движок**
1. Временно привязывает `this` внутри функции к `thisArg`.
2. Вызывает функцию немедленно.
3. После вызова контекст возвращается к обычному.
    
###### **Пример**
```js
function greet(greeting, punctuation) {
  console.log(`${greeting}, ${this.name}${punctuation}`);
}

const user = { name: 'Alice' };

greet.call(user, 'Hello', '!'); 
// Hello, Alice!
```
Здесь:
- `this` внутри `greet` = `{ name: 'Alice' }`
- Аргументы переданы через запятую.
