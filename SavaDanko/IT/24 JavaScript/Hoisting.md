**Hoisting** (подъём) — это механизм JavaScript, при котором объявления переменных и функций «поднимаются» в верхнюю часть своей области видимости во время фазы компиляции, до выполнения кода.

###### **Функции, объявленные через `function`**
Полностью поднимаются: и имя, и тело функции.
```js
sayHi(); // Привет!

function sayHi() {
  console.log("Привет!");
}
```

###### **Переменные, объявленные через `var`**
Поднимается только объявление, но не присваивание значения.
```js
console.log(a); // undefined
var a = 5;
```
Объявление `var a` поднимается, но `a = 5` — остаётся на месте. Поэтому `a` существует, но ещё не инициализирована.

###### **`let` и `const`**
Тоже поднимаются, но попадают в "временную мёртвую зону" (TDZ). Обращение до инициализации вызовет ошибку.
```js
console.log(b); // ReferenceError
let b = 10;
```
С `let` и `const` переменная технически уже известна движку, но доступ к ней запрещён до строки инициализации.
