`UIEvent` — это интерфейс DOM, унаследованный от `Event`, который описывает события пользовательского интерфейса, происходящие в окне просмотра (viewport).

###### **Наследование**
```
Event
  ↑
UIEvent
    ↑
    ├── MouseEvent
    ├── KeyboardEvent
    ├── FocusEvent
    ├── WheelEvent
    ├── InputEvent
    └── CompositionEvent
```
    
###### **Свойства `UIEvent`**
- `detail` — дополнительная информация о событии. Значение зависит от типа события (например, для `click` это количество последовательных кликов).
- `view` — объект `Window`, в котором произошло событие.
