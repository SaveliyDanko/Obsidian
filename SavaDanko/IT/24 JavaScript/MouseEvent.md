`MouseEvent` — это объект события, который создаётся браузером при взаимодействии пользователя с мышью, трекпадом или другим указывающим устройством.  
Он содержит всю информацию о событии: какой элемент был целью, где находился курсор, какие кнопки и модификаторы были нажаты, и т.д.

Наследуется от:
```
Event → UIEvent → MouseEvent
```

###### **Когда создаётся `MouseEvent`**
`MouseEvent` возникает при событиях:
- Кнопки мыши:
    - `click`
    - `dblclick`
    - `mousedown`
    - `mouseup`
    - `contextmenu` (правая кнопка)
        
- Движение мыши:
    - `mousemove`   
    - `mouseenter` / `mouseleave` (без всплытия)
    - `mouseover` / `mouseout` (с всплытием)

- События на колесе мыши (отдельный тип — `WheelEvent`, но тоже наследуется от `MouseEvent`).
    
###### **Основные свойства `MouseEvent`**
- `altKey` — возвращает `true`, если во время события была нажата клавиша ALT.
- `button` — указывает, какая кнопка мыши была нажата (0 — левая, 1 — средняя, 2 — правая).
- `buttons` — указывает, какие кнопки мыши были нажаты (битовая маска).
- `clientX` — координата X указателя мыши относительно окна браузера (viewport).
- `clientY` — координата Y указателя мыши относительно окна браузера.
- `ctrlKey` — возвращает `true`, если во время события была нажата клавиша CTRL.
- `detail` — дополнительная информация о событии (например, количество кликов подряд).
- `metaKey` — возвращает `true`, если во время события была нажата клавиша Meta (Windows key / Command ⌘ на Mac).
- `offsetX` — координата X указателя мыши относительно целевого элемента (`target`).
- `offsetY` — координата Y указателя мыши относительно целевого элемента.
- `pageX` — координата X указателя мыши относительно всей страницы (учитывает прокрутку).
- `pageY` — координата Y указателя мыши относительно всей страницы.
- `relatedTarget` — элемент, связанный с событием (например, при `mouseover` — элемент, с которого ушла мышь, а при `mouseout` — элемент, на который она пришла).
- `screenX` — координата X указателя мыши относительно экрана устройства.
- `screenY` — координата Y указателя мыши относительно экрана устройства.
- `shiftKey` — возвращает `true`, если во время события была нажата клавиша SHIFT.
