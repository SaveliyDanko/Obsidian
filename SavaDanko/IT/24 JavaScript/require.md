`require` — это функция, которая загружает и возвращает модуль (другой файл JS или библиотеку). Она позволяет разбивать код на отдельные файлы и использовать их повторно.

Пример:
```js
const fs = require('fs'); // Подключение встроенного модуля для работы с файлами

const data = fs.readFileSync('file.txt', 'utf8');
```
