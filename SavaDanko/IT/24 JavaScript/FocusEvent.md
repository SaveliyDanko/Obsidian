[[focus and blur]]
[[focusin and focusout]]

---
The FocusEvent Object handles events that occur when elements gets or loses focus.

###### **Свойства `FocusEvent`**
- `relatedTarget` — элемент, который связан с событием фокуса.
    - Для события `focusin` или `focus` — это элемент, с которого фокус ушёл.
    - Для события `focusout` или `blur` — это элемент, на который фокус переместился.
