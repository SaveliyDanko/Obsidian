`map` — контейнер, который хранит пары ключ — значение, где ключи уникальны и автоматически отсортированы.

#### Подключение и объявление
```cpp
#include <map>

map<string, int> age;
```

#### ==Основные методы==
`operator[]` - Доступ по ключу, вставляет `key` с default значением, если не найден

`at(key)` - доступ по ключу, выбрасывает исключение, если не найден

`insert({key, value})` - вставка пары, если ключа ещё нет

`earse(key)` - удаление по ключу

`find(key)` - возвращает итератор на элемент или `end()`

`count(key)` - вернёт 1, если ключ есть; иначе 0

`size()` - количество элементов

`clear` - oчистка всех элементов

`begin(), end()` - итераторы на начало и конец

#### ==Кастомная сортировка по ключу==
```cpp
struct ReverseCompare {
    bool operator()(const int& a, const int& b) const {
        return a > b;  // по убыванию
    }
};

std::map<int, std::string, ReverseCompare> m;
```
