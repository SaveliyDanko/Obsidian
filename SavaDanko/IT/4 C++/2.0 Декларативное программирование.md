**Декларативное программирование** — это стиль, при котором программист описывает желаемый результат, а не последовательность шагов для его достижения. 
Компилятор или интерпретатор сам определяет, как достичь результата.

###### **Примеры декларативного кода**
Императивный подход (пошаговые инструкции):
```pseudocode
if file does not exist on server:
    copy file to server
    verify file copied successfully
```

Декларативный подход (описание конечного состояния):
```pseudocode
ensure file is present on server
```

###### **Примеры языков**
SQL – запросы к базам данных
Prolog – логическое и функциональное программирование
React – декларативный UI с виртуальным DOM
