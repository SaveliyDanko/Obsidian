#### <mark style="background: #ABF7F7A6;">if ... else if ... else</mark>
```cpp
if (условие1) {
    // блок, если условие1 верно
} else if (условие2) {
    // если условие 1 ложно, проверяется следующее условие
} else if (условие3) {
    // и так далее...
} else {
    // если ни одно условие не выполнилось
}
```

#### <mark style="background: #ABF7F7A6;">Тернарный оператор `?:`</mark>
Для простых `if-else` можно использовать **тернарный оператор**.
```cpp
(условие) ? (если true) : (если false)
```

#### <mark style="background: #ABF7F7A6;">Логические операторы (`&&`, `||`, `!`)</mark>
Для объединения условий используют:
- `&&` (и) – **оба условия должны быть истинными**.
- `||` (или) – **хватит одного истинного условия**.
- `!` (не) – **инвертирует условие**.

#### <mark style="background: #ABF7F7A6;">switch</mark>
Если проверяется одно значение, лучше использовать `switch`.
```cpp
switch (переменная) {
    case значение1:
        // действия, если переменная == значение1
        break; // выход из switch

    case значение2:
        // действия
        break;

    default:
        // выполняется, если переменная не равна ни одному case
}
```
✔ `switch` работает **быстрее, чем `if-else`**, если вариантов много.
