#### <mark style="background: #ABF7F7A6;">1. Виды циклов в C++</mark>
В C++ есть три основных вида циклов:
1. `for` — используется, когда известно количество итераций.
2. `while` — выполняется, пока истинно условие.
3. `do-while` — выполняется хотя бы один раз.

#### <mark style="background: #ABF7F7A6;">2. Цикл for</mark>
Используется, когда заранее известно количество повторений.
```cpp
for (инициализация; условие; обновление) {
    // Тело цикла
}
```

#### <mark style="background: #ABF7F7A6;">3. Цикл while</mark>
Выполняется, пока условие **истинно** (`true`).
```cpp
while (условие) {
    // Тело цикла
}
```

#### <mark style="background: #ABF7F7A6;">4. Цикл do-while</mark>
Отличается от `while` тем, что **выполняется хотя бы один раз**, даже если условие ложно.
```cpp
do {
    // Тело цикла
} while (условие);
```

#### <mark style="background: #ABF7F7A6;">5. Операторы управления циклами</mark>
**`break` — прерывание цикла**
```cpp
for (int i = 0; i < 10; i++) {
    if (i == 5) break;
    std::cout << i << " ";
}
```

**`continue` — пропуск текущей итерации**
```cpp
for (int i = 0; i < 5; i++) {
    if (i == 2) continue;
    std::cout << i << " ";
}
```

#### <mark style="background: #ABF7F7A6;">6. Диапазонный цикл `for` (C++11)</mark>
Работает с массивами и контейнерами `vector`, `array`.

#### **Пример:**
```cpp
#include <iostream>
#include <vector>

int main() {
    vector<int> numbers = {1, 2, 3, 4, 5};

    for (int num : numbers) {
        cout << num << " ";
    }
    return 0;
}
```
