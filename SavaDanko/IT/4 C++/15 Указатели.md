#### <mark style="background: #ABF7F7A6;">Что такое указатель?</mark>
**Указатель** — это переменная, которая хранит **адрес памяти** другой переменной.
Проще говоря, указатель не хранит значение, он хранит место в памяти, где это значение лежит.

#### <mark style="background: #ABF7F7A6;">Как объявить указатель?</mark>
Для объявления указателя используется символ `*`:
```cpp
тип_данных* имя_указателя;
```

#### <mark style="background: #ABF7F7A6;">Оператор взятия адреса `&`</mark>
Чтобы получить адрес памяти переменной, используем оператор взятия адреса (`&`):
```cpp
int num = 5;
int* ptr = &num;  // ptr хранит адрес переменной num
```

#### <mark style="background: #ABF7F7A6;">Оператор разыменования `*`</mark>
Чтобы получить значение по указателю (т.е. обратиться к переменной, на которую указатель ссылается), используется оператор разыменования (`*`):
```cpp
int num = 10;
int* ptr = &num;

std::cout << *ptr;  // выводит 10
```

#### <mark style="background: #ABF7F7A6;">Указатели и массивы</mark>
Имя массива в C++ — это указатель на его первый элемент.

#### <mark style="background: #ABF7F7A6;">Арифметика указателей</mark>
Указатели поддерживают операции сложения и вычитания целых чисел.

#### <mark style="background: #ABF7F7A6;">Специальные виды указателей</mark>
#### ==1. Указатель на указатель (двойной указатель):==
```cpp
int a = 5;
int* ptr = &a;   // указатель на переменную
int** pptr = &ptr; // указатель на указатель

std::cout << **pptr; // выводит 5
```

#### ==2. Указатель `nullptr` (нулевой указатель)==
Всегда следует инициализировать указатель значением `nullptr`, если он не используется сразу:
```cpp
int* ptr = nullptr;

if (ptr == nullptr) {
    std::cout << "Указатель не инициализирован!\n";
}
```
