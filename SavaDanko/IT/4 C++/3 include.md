Директива `#include` в **C++** используется для **включения стандартных или пользовательских заголовочных файлов** в программу. Она обрабатывается **препроцессором** и указывает на вставку содержимого указанного файла в исходный код программы.

Этот процесс называется **включением файлов (File Inclusion)** и позволяет **использовать стандартные библиотеки** C++, а также **повторно использовать код**, написанный пользователем.

### <mark style="background: #ABF7F7A6;">Синтаксис include в C++</mark>
В C++ существует **два способа** подключения заголовочных файлов:
**Подключение стандартных заголовочных файлов через `< >`**
- Используется для **стандартных библиотек**, которые находятся в **системных директориях компилятора**.
- Пример:
```cpp
#include <iostream>  // Подключение стандартного заголовочного файла
```


**Подключение пользовательских заголовочных файлов через `""`**
- Используется для **файлов, созданных пользователем**.
- Пример:
```cpp
#include "my_header.h"  // Подключение пользовательского заголовочного файла
```  
- В этом случае препроцессор **сначала ищет файл в текущей директории** и, если не находит, обращается к стандартным путям поиска.
