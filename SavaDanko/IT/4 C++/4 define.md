### <mark style="background: #ABF7F7A6;">1. Что такое `#define` в C++?</mark>
Директива **`#define`** – это **препроцессорная директива** в C++, используемая для определения **макросов**. Макросы позволяют **заменять код или константные значения** на заданные фрагменты.  
Когда препроцессор встречает имя макроса в коде, он **заменяет его на соответствующее значение или выражение**, определённое в `#define`.

### <mark style="background: #ABF7F7A6;">2. Виды макросов в C++</mark>
В C++ с помощью `#define` можно определить **несколько видов макросов**:
1️⃣ **Константные макросы (Constant Macros)**  
2️⃣ **Цепочечные макросы (Chain Macros)**  
3️⃣ **Макросы-выражения (Macro Expressions)**  
4️⃣ **Многострочные макросы (Multiline Macros)**

### <mark style="background: #ABF7F7A6;">3. Константные макросы</mark>
```cpp
#define ИМЯ_МАКРОСА значение
```

### <mark style="background: #ABF7F7A6;">4. Цепочечные макросы (Chain Macros)</mark>
```cpp
#define МАКРОС1 значение1
#define значение1 конечное_значение
```

### <mark style="background: #ABF7F7A6;">5. Макросы-выражения (Macro Expressions)</mark>
```cpp
#define МАКРОС (выражение)
```

### <mark style="background: #ABF7F7A6;">6. Макросы-функции</mark>
```cpp
#define МАКРОС(параметры) (выражение)
```

### <mark style="background: #ABF7F7A6;">7. Многострочные макросы (Multiline Macros)</mark>
```cpp
#define МАКРОС значение \
                 значение2 \
                 значение3
```
