`string` — класс, который позволяет удобно работать с текстовыми данными.
```cpp
#include <string>
```

#### <mark style="background: #ABF7F7A6;">Создание и инициализация строк:</mark>
Есть несколько способов создания строк:
```cpp
string s1 = "Hello";        // из литерала
string s3(5, 'x');         // строка "xxxxx"
string s;                  // пустая строка ""
```

#### <mark style="background: #ABF7F7A6;">Основные операции со строками:</mark>
#### ==Конкатенация строк==
```cpp
std::string s1 = "Hello, ";
std::string s2 = "World!";
std::string result = s1 + s2;  // "Hello, World!"
```

#### ==Получение длины строки==
```cpp
std::string s = "Example";
std::cout << s.size(); // 7
```

#### <mark style="background: #ABF7F7A6;">Доступ к символам строки:</mark>
Индексация символов начинается с нуля.
```cpp
std::string s = "hello";
char c = s[1];      // 'e'
char ch = s.at(2);  // 'a'
```

Отличие метода `at()` от оператора `[ ]`:
- `at(index)` проверяет выход за пределы строки и генерирует исключение.
- `[]` не проверяет границы и быстрее, но менее безопасен.

#### <mark style="background: #ABF7F7A6;">Полезные методы std::string:</mark>

| Метод                    | Что делает                                   |
| ------------------------ | -------------------------------------------- |
| `empty()`                | Проверка, пустая ли строка                   |
| `clear()`                | Очищает строку (делает пустой)               |
| `substr(pos, len)`       | Возвращает подстроку                         |
| `find(str)`              | Ищет подстроку, возвращает индекс или `npos` |
| `replace(pos, len, str)` | Заменяет часть строки на другую              |

####  <mark style="background: #ABF7F7A6;">Сравнение строк:</mark>
Строки можно легко сравнивать, используя обычные операторы сравнения (`==`, `!=`, `<`, `>`, и т.д.)

#### <mark style="background: #ABF7F7A6;">Чтение строк:</mark>
Для ввода строки, включая пробелы, удобно использовать функцию:
```cpp
string s;
getline(cin, s);  // считывает строку целиком
```

Если в строке нет пробелов, то подойдёт и:
```cpp
string s;
cin >> s; // считывает до первого пробела
```

#### <mark style="background: #ABF7F7A6;">Важные нюансы работы с std::string:</mark>
- Строка хранит свои символы в динамической памяти (куча), что удобно, но медленнее работы с фиксированным массивом символов на стеке (`char[]`).
- Для передачи строки в функции используйте ссылки (`const string& s`), чтобы избежать лишнего копирования.
