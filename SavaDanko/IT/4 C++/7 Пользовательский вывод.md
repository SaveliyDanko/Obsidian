В языке C++ для вывода текста на экран используется поток **`std::cout`** из стандартной библиотеки `<iostream>`. 

#### <mark style="background: #ABF7F7A6;">Базовый вывод через `std::cout`</mark>
Стандартный поток вывода `std::cout` работает с оператором `<<` ("потоковый сдвиг влево"):
```cpp
#include <iostream>

int main() {
    std::cout << "Hello, world!" << std::endl;
    return 0;
}
```

**Эквивалентный вариант без `std::endl`**:
```cpp
std::cout << "Hello, world!\n";
```
`\n` предпочтительнее `std::endl`, потому что `std::endl` дополнительно сбрасывает буфер (`flush`), что может замедлить выполнение.
