#### <mark style="background: #ABF7F7A6;">Что такое vector?</mark>
- Контейнер, который автоматически управляет памятью.
- Хранит элементы в непрерывном участке памяти.
- Может автоматически увеличиваться и уменьшаться в размере при добавлении и удалении элементов.

#### <mark style="background: #ABF7F7A6;">Создание и инициализация вектора</mark>
```cpp
#include <vector>
vector<int> v;                                // пустой вектор int
vector<int> v1(5);                         // вектор из 5 элементов, инициализированных нулями
vector<int> v2(3, 100);                // [100, 100, 100]
vector<int> v3 = {1, 2, 3, 4};      // [1, 2, 3, 4]
```

#### <mark style="background: #ABF7F7A6;">Основные методы vector</mark>

| Метод         | Описание                                         |
| ------------- | ------------------------------------------------ |
| `push_back()` | Добавляет элемент в конец                        |
| `pop_back()`  | Удаляет последний элемент                        |
| `size()`      | Возвращает текущее количество элементов          |
| `empty()`     | Проверка, пуст ли вектор                         |
| `clear()`     | Удаляет все элементы                             |
| `resize(n)`   | Меняет размер вектора                            |
| `at(index)`   | Возвращает элемент по индексу с проверкой границ |
| `front()`     | Возвращает первый элемент                        |
| `back()`      | Возвращает последний элемент                     |

#### <mark style="background: #ABF7F7A6;">Итераторы</mark>
Итераторы позволяют перебирать элементы вектора:
```cpp
vector<int> v = {10, 20, 30};

for (int num : v) {
    cout << num << " ";
}
```
