#### <mark style="background: #ABF7F7A6;">1. Определение функции</mark>
Стандартный синтаксис функции:
```cpp
тип имя_функции(параметры) {
    // Тело функции
    return значение; // (если тип не void)
}
```

#### **Пример:**
```cpp
// Определение функции
int sum(int a, int b) {
    return a + b;
}
```

#### <mark style="background: #ABF7F7A6;">2. Типы функций</mark>
##### ==(1) Функции с возвращаемым значением==
```cpp
int multiply(int x, int y) {
    return x * y;
}
```

#### ==(2) Функции без возвращаемого значения (`void`)==
Функции `void` ничего не возвращают и используются для выполнения действий.
```cpp
void printMessage() {
    std::cout << "Привет, мир!" << std::endl;
}
```

#### ==(3) Функции с параметрами по умолчанию==
Можно задавать параметры, которые принимают значение по умолчанию.
```cpp
void greet(string name = "Гость") {
    cout << "Привет, " << name << "!\n";
}
```

#### ==(4) Перегрузка функций (Function Overloading)==
В C++ можно создавать несколько функций с одинаковым именем, но разными параметрами.
```cpp
int add(int a, int b) {
    return a + b;
}

double add(double a, double b) {
    return a + b;
}
```
В зависимости от переданных аргументов компилятор выберет нужную функцию.

#### ==(5) Рекурсивные функции==
Функция вызывает саму себя, пока не выполнится базовое условие.
```cpp
int factorial(int n) {
    if (n == 0) return 1;
    return n * factorial(n - 1);
}
```


#### <mark style="background: #ABF7F7A6;">3. Передача параметров</mark>
В C++ параметры можно передавать **по значению, по ссылке и по указателю**.

#### ==(1) Передача по значению (копия)==
Функция получает копию аргумента и не изменяет оригинал.
```cpp
void changeValue(int x) {
    x = 10; // Изменится только локальная копия
}
```

#### ==(2) Передача по ссылке (изменяет оригинал)==
Функция получает **ссылку на переменную** и может её менять.
```cpp
void changeValue(int &x) {
    x = 10; // Изменит оригинальную переменную
}
```

```cpp
int a = 5;
changeValue(a);
std::cout << a; // 10
```

#### ==(3) Передача по указателю==
Функция получает **указатель на переменную** и изменяет её через `*`.
```cpp
void changeValue(int *x) {
    *x = 10;
}
```

```cpp
int a = 5;
changeValue(&a);
std::cout << a; // 10
```

#### <mark style="background: #ABF7F7A6;">4. Инлайн-функции (inline)</mark>
Инлайн-функции подставляют код прямо в место вызова, снижая накладные расходы вызова функции.
```cpp
inline int square(int x) {
    return x * x;
}
```
Используется для **простых коротких функций**.

#### <mark style="background: #ABF7F7A6;">5. Шаблонные функции (Generic Functions)</mark>
Позволяют писать универсальные функции, работающие с разными типами.
```cpp
template <typename T>
T maxValue(T a, T b) {
    return (a > b) ? a : b;
}
```

```cpp
std::cout << maxValue(10, 20); // 20
std::cout << maxValue(5.5, 2.3); // 5.5
```
