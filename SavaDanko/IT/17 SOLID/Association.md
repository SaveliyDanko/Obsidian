###### **Ассоциация**
Ассоциация — это отношение между двумя классами, при котором один объект использует другой без жёсткой зависимости.

###### **Агрегация**
Агрегация — это частный случай ассоциации, который отражает отношение целое–часть.
- Has-a (имеет) отношение: одно целое содержит объекты других классов.
- Части могут существовать независимо от целого.
- Жизненные циклы объектов не связаны: если целое уничтожено, части продолжают жить.
```java
class Teacher {
    String name;
    Teacher(String name) { this.name = name; }
}

class Student {
    String name;
    Teacher teacher; // Агрегация: студент связан с учителем

    Student(String name, Teacher teacher) {
        this.name = name;
        this.teacher = teacher;
    }
}
```

###### **Композиция**
Композиция — это жёсткий вариант ассоциации, когда один объект является частью другого и не может существовать отдельно.
- Отношение part-of (часть целого).
- Жизненный цикл общий: если целое уничтожается, уничтожаются и его части.
- Сильная связь: часть не может существовать без владельца.
```java
class Engine {
    void start() {
        System.out.println("Двигатель запущен");
    }
}

class Car {
    private Engine engine; // композиция

    Car() {
        this.engine = new Engine(); // двигатель создаётся вместе с машиной
    }

    void drive() {
        engine.start();
        System.out.println("Машина поехала");
    }
}
```
