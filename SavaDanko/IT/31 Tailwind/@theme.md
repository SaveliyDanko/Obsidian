---
Переменные темы — это специальные CSS-переменные, определяемые с помощью директивы `@theme`, которые влияют на то, какие утилитарные классы будут доступны в вашем проекте.

Вы можете добавить новый цвет или шрифт в проект, определив переменную темы, например:
```css
@theme {
  --color-mint-500: oklch(0.72 0.11 178);
  --font-serif: ui-serif, Georgia, Cambria, "Times New Roman", Times, serif;
}
```

---
Некоторые переменные темы используются не для создания утилитарных классов, а для определения вариантов (variants).

Например, переменные темы в пространстве имён `--breakpoint-*` определяют, какие адаптивные (responsive) брейкпоинты существуют в вашем проекте:
```css
@theme {
  --breakpoint-3xl: 120rem;
}
```

---
Переменные темы определяются в пространствах имён (namespaces),  
и каждое пространство соответствует одному или нескольким API для утилитарных классов или вариантов.

Определяя новые переменные темы в этих пространствах, вы автоматически получаете доступ к новым утилитарным классам и вариантам в проекте.

| Пространство имён | Соответствующие утилитарные классы                                         |
| ----------------- | -------------------------------------------------------------------------- |
| `--color-*`       | Цветовые утилиты, например `bg-red-500`, `text-sky-300` и множество других |
| `--font-*`        | Утилиты для семейства шрифтов, например `font-sans`                        |
| `--text-*`        | Утилиты для размера шрифта, например `text-xl`                             |
| `--font-weight-*` | Утилиты для толщины шрифта, например `font-bold`                           |
| `--tracking-*`    | Утилиты для межбуквенного интервала, например `tracking-wide`              |
| `--leading-*`     | Утилиты для межстрочного интервала, например `leading-tight`               |
| `--breakpoint-*`  | Адаптивные варианты, например `sm:*`                                       |
| `--spacing-*`     | Утилиты отступов и размеров, например `px-4`, `max-h-16` и другие          |
| `--ease-*`        | Утилиты функций временной кривой анимации, например `ease-out`             |
| `--animate-*`     | Утилиты анимации, например `animate-spin`                                  |

---
**Переменные темы по умолчанию (Default theme variables)**
Когда вы импортируете `tailwindcss` в начале вашего CSS-файла, фреймворк автоматически подключает набор стандартных (default) переменных темы, чтобы вы могли сразу начать работу без ручной настройки.
