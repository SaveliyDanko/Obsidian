[@font-face](https://developer.mozilla.org/en-US/docs/Web/CSS/@font-face)

---
`@font-face` — это директива CSS, с помощью которой можно подключать пользовательские шрифты, используемые для отображения текста. Шрифт может быть загружен как с удалённого сервера, так и с компьютера пользователя, если он уже установлен локально.
```css
@font-face {  
    font-family: 'Example';  
    src: url('src/assets/fonts/Example.woff2') format('woff2');  
}
```

###### **Дескрипторы**
`font-family`
Задаёт имя, которое будет использоваться в качестве значения свойства `font-family`. Это имя является обязательным для корректности правила `@font-face`.

---
`font-style`
Определяет стиль начертания шрифта. Может принимать одно значение или диапазон значений, например: `font-style: oblique 20deg 50deg;`

---
`font-weight`
Определяет толщину начертания. Может принимать одно значение или диапазон, например: `font-weight: 100 400;`

---
`src`
Указывает источники (ресурсы) шрифта, включая подсказки о его формате и технологии. Параметр `src` обязателен для корректного описания `@font-face`.
