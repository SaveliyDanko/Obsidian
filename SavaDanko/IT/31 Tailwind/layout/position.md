[position](https://cssreference.io/property/position/)

---
###### **static**
```css
position: static;    /* поведение по умолчанию */
```
- Элемент располагается в общем потоке страницы, строго по порядку, без смещения.
- Свойства смещения (`top`, `left`, `right`, `bottom`) не работают.
- Элемент не «позиционирован», то есть не может быть родителем для абсолютно позиционированных (`absolute`) потомков.

###### **relative**
```css
position: relative;
```
- Элемент остаётся в общем потоке, но его можно сдвинуть относительно своего обычного положения с помощью `top`, `left`, `right`, `bottom`.
- При этом его место в потоке сохраняется — другие элементы ведут себя так, будто он на месте.
- Элемент становится позиционированным контейнером для вложенных элементов с `position: absolute`.

###### **absolute**
```css
position: absolute;
```
- Элемент выходит из естественного потока документа.
- Он позиционируется относительно ближайшего позиционированного предка.
- Так как элемент является позиционированным, он сам может служить опорной точкой для других абсолютно позиционированных элементов.

###### **fixed**
```css
position: fixed;
```
- Элемент выходит из естественного потока документа.
- Он позиционируется относительно области просмотра (viewport).
- Поскольку элемент является позиционированным, он также может служить опорной точкой для абсолютно позиционированного другого блока.
