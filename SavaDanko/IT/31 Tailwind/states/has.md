Селектор `:has()` в CSS — это мощный псевдокласс, который позволяет выбирать элемент в зависимости от наличия (или содержимого) его потомков.
Его часто называют “родительским селектором”, потому что он даёт возможность выбирать родителя на основе содержимого.

###### **Синтаксис**
```css
A:has(B)
```
— выбирает элемент `A`, если внутри него (на любом уровне вложенности) есть элемент `B`.

---
###### **Примеры**
Выделить карточки, где есть картинка
```css
.card:has(img) {
  border: 2px solid green;
}
```
Все `.card`, содержащие `<img>`, получат зелёную рамку.

---
Стилизация родителя, если внутри есть активная кнопка
```css
.form:has(button:focus) {
  outline: 2px solid blue;
}
```
Если кнопка внутри формы в фокусе, подсвечивается вся форма.

---
Выделить пустые контейнеры (нет `.item`)
```css
.container:not(:has(.item)) {
  background: #f9dada;
}
```
Подсвечивает контейнеры без элементов `.item`.
