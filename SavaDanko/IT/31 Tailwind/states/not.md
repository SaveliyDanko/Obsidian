---
Псевдокласс `:not()` позволяет исключить из выборки элементы, соответствующие определённому селектору.

###### **Синтаксис**
```css
селектор:not(условие)
```
Выбирает все элементы, соответствующие `селектор`, кроме тех, что соответствуют `условие`.

###### **Примеры**
Исключить определённый класс
```css
button:not(.primary) {
  background: #ccc;
}
```
Все кнопки кроме `.primary` будут серыми.

---
Исключить элемент по типу
```css
p:not(:first-child) {
  margin-top: 1em;
}
```
Отступ добавится ко всем абзацам, кроме первого.

---
Исключить несколько вариантов
С CSS4 можно использовать список селекторов в `:not()`:
```css
div:not(.active, .disabled, .hidden) {
  opacity: 1;
}
```
Селектор выберет все `div`, которые не имеют ни одного из указанных классов.
