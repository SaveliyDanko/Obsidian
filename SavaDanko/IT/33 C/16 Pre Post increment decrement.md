###### Prefix (pre): `++x`, `--x`
- Сначала изменяет, потом возвращает новое значение.
```c
int x = 5;
int a = ++x;   // x=6, a=6
```

###### Postfix (post): `x++`, `x--`
- Сначала возвращает старое значение, потом изменяет.
```c
int x = 5;
int a = x++;   // a=5, x=6
```
