В `C` ключевое слово `auto` — это storage class specifier (спецификатор класса хранения), и означает автоматическую длительность хранения: переменная живёт только внутри блока (обычно внутри функции) и уничтожается при выходе из него.

`auto` можно писать только для локальных переменных (block scope):
```c
void f(void) {
    auto int x = 10;  // то же самое, что просто: int x = 10;
}
```
Фактически `auto` в C почти никогда не пишут, потому что по умолчанию любая локальная переменная без `static` — уже auto.
