###### **Импорты на этапе сборки**
Tailwind автоматически объединяет другие CSS-файлы, подключённые с помощью `@import`, без необходимости использования отдельных инструментов для препроцессинга.

Пример (`app.css`):
```css
@import "tailwindcss";
@import "./typography.css";
```
В этом примере файл `typography.css` будет включён в итоговый скомпилированный CSS с помощью Tailwind, без применения дополнительных инструментов вроде Sass или `postcss-import`.
