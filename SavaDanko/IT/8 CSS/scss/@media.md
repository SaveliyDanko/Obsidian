###### **`@media` — медиазапросы (адаптивный дизайн)**
Позволяет написать стили, которые применяются только при определённых условиях, например ширина экрана. Это CSS-фича, которая активно используется в SCSS, особенно с вложенностью.

###### **Пример:**
```scss
.container {
  width: 100%;

  @media (min-width: 768px) {
    width: 750px;
  }

  @media (min-width: 1200px) {
    width: 1140px;
  }
}
```

Может работать в сочетании с переменными:
```scss
$breakpoint: 768px;

.card {
  @media (min-width: $breakpoint) {
    padding: 2rem;
  }
}
```
