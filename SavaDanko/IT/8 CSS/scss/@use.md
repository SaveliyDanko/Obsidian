###### **`@use` — подключение модулей**
- Загружает другой SCSS-файл один раз (даже при множественных `@use`)
- Изолирует переменные, миксины и функции: доступ к ним идёт через псевдоним (namespace)
- Предотвращает конфликты имён
    

###### **Пример:**
```scss
// _colors.scss
$primary: #3498db;
```

```scss
// style.scss
@use 'colors';

.button {
  background-color: colors.$primary;
}
```

Можно задать псевдоним:
```scss
@use 'colors' as c;

.button {
  background-color: c.$primary;
}
```

Можно отключить псевдоним (не рекомендуется, но возможно):
```scss
@use 'colors' as *; // все переменные доступны без префикса
```
