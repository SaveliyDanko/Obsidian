The `scroll-behavior` property in CSS controls the _scrolling animation_ when users navigate through an element.
```css
scroll-behavior: auto | smooth | inherit | initial | unset;
```

| Value     | Description                                                            |
| --------- | ---------------------------------------------------------------------- |
| `auto`    | Default. Instantly jumps to the target location without any animation. |
| `smooth`  | Animates the scroll to smoothly transition to the target location.     |
| `inherit` | Inherits the value from its parent.                                    |
| `initial` | Resets to the default (`auto`).                                        |
| `unset`   | Resets the property depending on inheritance rules.                    |
