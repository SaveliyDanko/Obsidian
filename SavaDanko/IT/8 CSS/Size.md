#css

---
###### **Width и Height** — управляют шириной и высотой элемента.
```css
width: 100px;
height: 200px;
```

###### **Padding** — определяет расстояние между содержимым элемента и его границей.
```css
padding: 10px;
```

###### **Margin** — задаёт отступ снаружи границ элемента.
```css
margin: 20px;
```

###### **Border** — определяет толщину рамки вокруг элемента.
```css
border: 2px solid black;
```


#### ==Единицы измерения в CSS==
###### Абсолютные единицы — фиксированные значения, не зависят от других элементов.
`px` (пиксели): самая распространённая абсолютная единица.
```css
width: 300px;
```
        
###### Относительные единицы — зависят от других свойств в документе.
`%` (проценты): относительно размера родительского элемента.
```css
width: 50%;
```
  
`em`: относительно размера шрифта самого элемента.
```css
padding: 1.5em;
```
  
`rem`: относительно размера шрифта корневого элемента (обычно `<html>`).
```css
margin: 2rem;
```

###### **Единицы, основанные на размере области просмотра (viewport)**:
`vw` (ширина окна просмотра): 1% от ширины окна браузера.
```css
width: 50vw;
```

`vh` (высота окна просмотра): 1% от высоты окна браузера.
```css
height: 50vh;
```

###### Единицы гибкой раскладки (Flex и Grid):
`fr` (доля): используется в сетках CSS Grid для распределения пространства.
```css
grid-template-columns: 1fr 2fr;
```
