`any` отключает систему типов — можно присвоить и использовать как угодно.

Пример:
```ts
let obj: any = { x: 0 };

// Ни одна из строк ниже не вызовет ошибок компилятора.
// Использование `any` отключает всю дальнейшую проверку типов.
obj.foo();
obj();
obj.bar = 100;
obj = "hello";
const n: number = obj;

```
Особенности:
- `any` — и подтип, и супертип всего
- Опасен: ломает type-safety
    
Best practice:  
Избегать в продакшн-коде. Лучше использовать `unknown`.
