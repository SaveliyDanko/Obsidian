1. **GET**
    - Получение ресурса.        
    - Параметры передаются через query string (`?id=10`).
    - Тело запроса обычно отсутствует.
        
2. **POST**
    - Создание нового ресурса или выполнение операции.
    - Данные передаются в теле запроса (JSON, XML, формы).
        
3. **PUT**
    - Полная замена ресурса. 
    - Тело запроса содержит новое состояние ресурса.
        
4. **PATCH**
    - Частичное обновление ресурса.
    - Передаются только изменённые поля.

5. **DELETE**    
    - Удаление ресурса.
