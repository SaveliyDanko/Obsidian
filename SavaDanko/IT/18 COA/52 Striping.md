**Чередование (Striping)** — метод организации данных, при котором поток информации разбивается на блоки фиксированного размера и записывается на несколько дисков последовательно.
- Strip (сегмент): Логический блок данных на одном физическом диске.
- Stripe (полоса): Совокупность сегментов, имеющих одинаковые адреса на всех дисках массива.
- Stripe Width (ширина): Количество физических дисков, участвующих в распределении данных.
- Stripe Size (размер полосы): Общий объем данных в одной полосе. Рассчитывается как: `размер сегмента * количество дисков`.
