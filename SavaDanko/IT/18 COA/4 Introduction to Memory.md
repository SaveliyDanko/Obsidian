## Text Elements
DRAM (Dynamic Random Access Memory)
- Storage Method: Stores data in capacitors that need periodic refreshing.
- Speed: Slower compared to SRAM.
- Power Consumption: Consumes more power due to constant refreshing.
- Cost: Cheaper than SRAM.
- Usage: Main memory (e.g., system RAM) in computers.
- Volatility: Volatile (loses data when powered off). ^JMjOVPiP

SRAM (Static Random Access Memory)
- Storage Method: Uses flip-flops to store data, no need for refreshing.
- Speed: Faster access times than DRAM.
- Power Consumption: Lower power usage when idle, but generally higher during access.
- Cost: More expensive due to complexity.
- Usage: Cache memory in CPUs and high-speed registers.
- Volatility: Volatile (loses data when powered off). ^HYS5EGUX

Cache ^6tuvFIf8

Secondary Memory is non-volatile storage used to store data and programs permanently. 
Unlike primary memory (like RAM), it does not lose data when the computer is turned off.
- Non-volatile: Retains data even without power.
- Capacity: Generally much larger than primary memory.
- Speed: Slower access compared to primary memory.
- Cost: Cheaper per unit of storage compared to RAM.
- Usage: Stores operating systems, software, user files, and backup data.
