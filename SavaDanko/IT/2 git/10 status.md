Показывает текущее состояние рабочего каталога и индекса — какие файлы изменены, какие подготовлены к коммиту, какие не отслеживаются и т.д. 

| Флаг               | Назначение                                                                                                   |
| ------------------ | ------------------------------------------------------------------------------------------------------------ |
| `-s` или `--short` | Краткий вывод: отображает статус файлов в виде двухбуквенных кодов (удобно для скриптов и командной строки). |
| `-b`               | Показывает текущую ветку (часто используется с `-s` как `git status -sb`).                                   |
| `--ignored`        | Показывает также файлы, игнорируемые `.gitignore`.                                                           |

#### Особенности:
- Показывает **текущее состояние файлов**:
    - изменения, не добавленные в индекс (Modified),
    - изменения, добавленные в индекс (Staged),
    - неотслеживаемые файлы (Untracked),
    - игнорируемые файлы (если запрошены).
