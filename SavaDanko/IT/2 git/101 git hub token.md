**Токен доступа GitHub** (Personal Access Token) — это ключ, который используется для аутентификации вместо пароля при доступе к GitHub из сторонних инструментов (например, `git`, `gh`, CI/CD).

С 2021 года GitHub запретил использовать обычные пароли для доступа к репозиториям через HTTPS (например, при `git push`, `git clone`), чтобы повысить безопасность. Вместо пароля теперь используется токен.
