Команда `git rm` удаляет файлы из рабочей директории и из области индексирования, подготавливая их к удалению в следующем коммите. 

| Флаг                 | Назначение                                                                                                                           |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| `--cached`           | Удаляет файл **только из индекса**, оставляя его в рабочей директории. Используется для **удаления из отслеживания**, но не с диска. |
| `-f` (или `--force`) | Принудительное удаление. Необходим, если файл уже изменён или удалён вручную.                                                        |
