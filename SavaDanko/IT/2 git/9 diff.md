Команда `git diff` показывает различия между рабочей директорией, индексом и коммитами. Это один из главных инструментов для анализа изменений до коммита или между коммитами.

| Команда / Флаг                     | Что сравнивает                              |
| ---------------------------------- | ------------------------------------------- |
| `git diff`                         | Рабочая копия ↔️ индекс                     |
| `git diff --cached` или `--staged` | Индекс ↔️ последний коммит                  |
| `git diff HEAD`                    | Рабочая копия + индекс ↔️ последний коммит  |
| `git diff <commit1> <commit2>`     | Различия между двумя коммитами              |
| `git diff <branch1> <branch2>`     | Различия между ветками                      |
| `git diff --name-only`             | Только имена изменённых файлов              |
| `git diff --stat`                  | Краткая статистика изменений (кол-во строк) |
| `git diff --color`                 | Подсветка изменений (по умолчанию включена) |
