`git branch` - просмотр существующих веток
`git branch -v` - просмотр последних коммитов на каждой ветке
`git branch <name>` -  создание новой ветки
`git branch -d <name>` - удаление ветки

`git branch --move <old-name> <new-name>` - переименование локальной ветки
`git push origin --delete <old-name>` - Удаление старой ветки на удалённом репозитории
`git push origin <new-name>` - Отправка новой ветки на удалённый репозиторий
`git push --set-upstream origin <new-name>` - Настройка отслеживания новой ветки
