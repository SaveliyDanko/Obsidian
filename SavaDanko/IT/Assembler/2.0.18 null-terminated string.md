Нуль-терминированная строка (null-terminated string) — это последовательность символов в памяти, которая заканчивается специальным символом — **нулевым байтом** (с кодом 0 или `'\0'`). Этот нулевой байт сигнализирует о конце строки.

### ==Основные характеристики:==
1. **Последовательность байтов**: Строка представляет собой массив байтов, где каждый байт соответствует символу строки в соответствии с кодировкой (например, ASCII).
2. **Завершающий символ**: Строка заканчивается байтом с кодом 0, что позволяет программам, работающим с такими строками, определить, где она заканчивается. Это ключевая особенность, которая отличает нуль-терминированные строки от других строковых форматов.
3. **Использование в C и подобных языках**: В языках программирования, таких как C, строки обычно реализованы как нуль-терминированные. Например, в C строка `"Hello"` хранится как следующие байты: `'H'`, `'e'`, `'l'`, `'l'`, `'o'`, `'\0'`.

### ==Пример нуль-терминированной строки:==
```c
char str[] = "Hello";
```

Этот код создаст строку, которая будет представлена в памяти так:
```
H  e  l  l  o  \0
```
