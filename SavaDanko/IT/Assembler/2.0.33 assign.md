Директива `%assign` в ассемблере используется для назначения **числовых значений** макросам. В отличие от `%define`, которая может определять макросы, содержащие строки или выражения, `%assign` работает только с **целыми числами** и позволяет присваивать числовые значения макросам или символам.

### ==Основные характеристики `%assign`:==
1. **Операции с числами**: `%assign` используется исключительно для работы с числовыми значениями и поддерживает целые числа. Это позволяет назначать числовые значения переменным и затем использовать их в выражениях.
2. **Мгновенное вычисление**: `%assign` вычисляет значение выражения сразу же в момент задания, а не в момент использования, как это делает `%define`.
3. **Применение в арифметических выражениях**: Макросы, определённые с помощью `%assign`, можно использовать в арифметических операциях, таких как сложение, умножение и т.д.

### ==Синтаксис:==
```asm
%assign имя значение
```
- **имя** — это макрос или символ, которому присваивается числовое значение.
- **значение** — это выражение, которое должно быть вычислено и присвоено макросу.

### ==Пример использования:==
```asm
%assign value 10     ; Присваиваем макросу value значение 10
%assign multiplier 3  ; Присваиваем макросу multiplier значение 3

mov rax, value       ; rax = 10
imul rax, multiplier ; rax = rax * 3 = 30
```
В этом примере макросы `value` и `multiplier` хранят числовые значения, которые потом используются в инструкции `mov` и арифметической операции `imul`.

### ==Важные моменты:==
1. **Вычисление при присвоении**: `%assign` вычисляет результат выражения сразу при его использовании. Это значит, что если переменные, использованные в выражении, изменятся после вызова `%assign`, это не повлияет на результат. Например:
    ```asm
    %define i 10
    %assign result i * 2   ; result = 20
    %define i 5
    mov rax, result        ; rax будет равен 20, так как result вычислился раньше
    ```
    Здесь значение `i` изменилось, но макрос `result` всё равно остаётся равным 20, так как оно было вычислено в момент вызова `%assign`.

2. **Работа с числовыми выражениями**: `%assign` позволяет использовать любые корректные числовые выражения:
    ```asm
    %assign num 5 + 3
    %assign result num * 2
    mov rax, result   ; rax = (5 + 3) * 2 = 16
    ```

3. **Ограничение на тип данных**: `%assign` работает только с целыми числами. Для строк или более сложных выражений следует использовать `%define`.

### ==Пример использования для счётчика:==
Директива `%assign` часто используется для задания счётчиков в циклах или для автоматического увеличения числовых значений.
```asm
%assign count 0   ; Инициализация счётчика

loop:
    ; Выполнение каких-то инструкций
    inc rax
    %assign count count + 1   ; Увеличение счётчика

    cmp count, 10             ; Сравнение счётчика с 10
    jne loop
```
