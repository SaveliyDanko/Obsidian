**`curl`** — это командная утилита для отправки HTTP-запросов из терминала.  
Название расшифровывается как: Client URL.

С помощью `curl` ты можешь:
- получать веб-страницы;
- взаимодействовать с API (например, REST API);
- скачивать файлы;
- отправлять формы;
- делать POST, PUT, DELETE-запросы и многое другое.

| Параметр      | Что делает                                                   |
| ------------- | ------------------------------------------------------------ |
| `-X`          | Метод запроса (`GET`, `POST`, `PUT`, `DELETE`)               |
| `-H`          | Добавляет заголовок (`Content-Type`, `Authorization` и т.п.) |
| `-d`          | Передаёт тело запроса (чаще всего JSON)                      |
| `-o filename` | Сохраняет ответ в файл                                       |
| `-O`          | Сохраняет как на сервере                                     |
| `-i`          | Показывает заголовки ответа                                  |
| `-v`          | Подробный вывод (debug: запрос, ответ, заголовки)            |
| `-u`          | Basic Auth: `curl -u user:pass https://...`                  |

```bash
curl -X GET "https://api.tracker.yandex.net/v2/priorities" \
  -H "Authorization: OAuth AQAAAABCD..." \
  -H "X-Org-ID: 1234567"
```
