#linux 

---
Команда `lsmod` показывает все загруженные в ядро модули. Это фактически обёртка над `/proc/modules`.

```bash
$ lsmod
Module                                 Size                      Used by
snd_hda_codec_hdmi          73728                    1
i915                                      299008                  2
xhci_pci                               20480                    0
```

| Колонка     | Значение                                                      |
| ----------- | ------------------------------------------------------------- |
| **Module**  | Имя модуля (`.ko` файл)                                       |
| **Size**    | Размер модуля в памяти (в байтах)                             |
| **Used by** | Сколько других модулей используют его (не процессов!) и какие |
