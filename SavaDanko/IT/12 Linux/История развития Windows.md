История Windows — это путь от графической надстройки над DOS до доминирующей операционной системы для персональных компьютеров. В отличие от Linux, Windows развивалась как закрытый коммерческий продукт компании Microsoft.


###### **1. Эпоха Оболочек (1985–1994)**
В это время Windows не была полноценной ОС, а лишь графическим интерфейсом для MS-DOS.
- Windows 1.0 (1985): Первая попытка отойти от командной строки. Окна нельзя было перекрывать (они располагались только плиткой).
- Windows 3.0 / 3.1 (1990–1992): Первый крупный успех. Появились «Диспетчер программ», поддержка шрифтов TrueType и знаменитая игра «Сапер». Система стала массовой.

###### **2. Слияние с потребительским рынком (1995–2000)**
- Windows 95: Революционный продукт. Появились кнопка «Пуск», панель задач и поддержка технологии Plug and Play. Система перешла на 32-битную архитектуру.
- Windows 98: Улучшенная версия 95-й с глубокой интеграцией Internet Explorer, что привело к знаменитым антимонопольным судам против Microsoft.

###### **3. Переход на ядро NT (Новая эра)**
Параллельно с домашними версиями Microsoft развивала профессиональную ветку Windows NT (New Technology), которая была гораздо стабильнее.
- Windows XP (2001): Легендарная система, объединившая домашнюю и профессиональную ветки на стабильном ядре NT. Она оставалась популярной более 10 лет.
- Windows Vista (2006): Попытка внедрить тяжелую графику (Aero) и усилить безопасность. Была встречена критикой за медлительность и навязчивый контроль учетных записей (UAC).
- Windows 7 (2009): «Работа над ошибками» Vista. Стала эталоном стабильности и удобства, до сих пор считается одной из лучших версий в истории.

###### **4. Современный этап: Облака и Сервисы (2012 — н.в.)**
- Windows 8 / 8.1: Радикальный эксперимент с «плиточным» интерфейсом для планшетов. Кнопку «Пуск» убрали, что вызвало гнев пользователей.
- Windows 10 (2015): Возврат к классическому меню «Пуск» и переход на модель «ОС как сервис» (постоянные обновления вместо выпуска новых версий раз в три года).
- Windows 11 (2021): Современный дизайн, обновленное меню «Пуск» в центре и фокус на интеграцию с AI (Copilot).
