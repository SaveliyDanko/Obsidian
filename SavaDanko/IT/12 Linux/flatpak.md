#linux 

---
`Flatpak` — это как "универсальный контейнер для десктоп-приложений", похожий на `snap`, но с гибкой системой прав и открытой архитектурой.

###### **Flatpack commands**
**1** `flatpak install flathub <app>` - Установить приложение из FlatHub

**2** `flatpak run <app>` - Запустить приложение

**3** `flatpak list` - Список установленных flatpak-приложений

**4** `flatpak update` - Обновить приложения

**5** `flatpak remove <app>` - Удалить приложение

**6** `flatpak info <app>` - Информация о приложении


###### **FlatHub — основной репозиторий приложений**
Flatpak использует `FlatHub` как главный магазин приложений:  
[https://flathub.org](https://flathub.org/)
