#linux 

---
**SSH (Secure Shell)** — это сетевой протокол и набор утилит, предназначенных для безопасного удалённого доступа к компьютерам и серверам по сети. Он обеспечивает зашифрованный канал связи, защищая данные от перехвата, подмены и атак типа man-in-the-middle.

###### Основные возможности SSH
1. Удалённый доступ к командной строке  
    ```bash
    ssh user@remote_host
    ```
    
2. Передача файлов с помощью SCP или SFTP:
    ```bash
    scp file.txt user@remote_host:/home/user/
    sftp user@remote_host
    ```
    

3. Аутентификация с помощью ключей, а не паролей:    
    - Генерация ключей:
        ```bash
        ssh-keygen -t rsa -b 4096
        ```
        
    - Копирование публичного ключа:
        ```bash
        ssh-copy-id user@remote_host
        ```
        
4. SSH-агент для хранения приватных ключей в памяти:
    ```bash
    eval "$(ssh-agent)"
    ssh-add ~/.ssh/id_rsa
    ```
