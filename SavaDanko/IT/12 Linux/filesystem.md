#linux #filesystem

**Файловая система (filesystem)** — это способ организации и хранения данных на носителе (жёсткий диск, флешка, SSD, SD-карта и т.п.).  
Она определяет:
- как файлы размещаются
- как ведётся учёт папок и путей
- какие есть ограничения по размеру файлов и накопителя.

**FAT32 (File Allocation Table 32)** - используется для EFI System Partition (ESP) $\to$ [[ESP]]

**ext4 (Fourth Extended Filesystem)** - современный стандарт для Linux-систем

**NTFS (New Technology File System)** - современный стандарт для Windows-систем

**exFAT (Extended FAT)** - флешки и SD-карты
