#linux 


---
Команда `ss` (от **socket statistics**) — это современный и быстрый инструмент в Linux для отображения информации о сетевых соединениях, открытых портах, прослушивающих сокетах и использующих их процессах.
```bash
ss [опции/ключи]
```

| Ключи | Значение                                                          |
| ----- | ----------------------------------------------------------------- |
| `-t`  | Показывать только TCP соединения                                  |
| `-u`  | Показывать только UDP соединения                                  |
| `-l`  | Показывать только прослушивающие сокеты (LISTEN)                  |
| `-n`  | Показывать IP-адреса и номера портов, а не имена (ускоряет вывод) |
| `-p`  | Показывать процесс (PID/имя), который использует сокет            |
| `-a`  | Показывать **все** сокеты (и активные соединения, и слушающие)    |
| `-r`  | Показывать маршруты                                               |
| `-s`  | Статистика по сокетам                                             |
| `-4`  | Только IPv4                                                       |
| `-6`  | Только IPv6                                                       |
| `-x`  | Показывать Unix-сокеты (локальные IPC)                            |
