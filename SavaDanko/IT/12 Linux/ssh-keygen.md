`ssh-keygen` — это стандартная утилита для создания, управления и конвертации криптографических ключей для SSH (Secure Shell) в системах Linux, Unix и macOS.



###### **Базовый синтаксис** `ssh-keygen`
```bash
ssh-keygen [опции]
```

Генерация ключа по умолчанию:
```bash
ssh-keygen
```
- Система спросит путь сохранения (по умолчанию: `~/.ssh/id_rsa`)
- Можно задать парольную фразу для дополнительной защиты приватного ключа (рекомендуется)
- Генерируется пара файлов:
    - `id_rsa` — приватный ключ
    - `id_rsa.pub` — публичный ключ

| Опция | Описание                             |
| ----- | ------------------------------------ |
| `-t`  | Тип ключа (rsa, ed25519, ecdsa, dsa) |
| `-b`  | Размер ключа в битах                 |
| `-f`  | Указать путь к файлу ключа           |
| `-C`  | Комментарий к ключу                  |
| `-N`  | Указать парольную фразу в команде    |

###### **Примеры ключей**
Современный и безопасный ключ Ed25519:
```bash
ssh-keygen -t ed25519 -f ~/.ssh/id_rsa_server_name -C "mykey for GitHub"
```

Классический RSA-ключ 4096 бит:
```bash
ssh-keygen -t rsa -b 4096 -f ~/.ssh/id_rsa_server_name -C "my server key"
```

---
**Права для ключей:**

| Файл / папка                             | Рекомендуемые права | Комментарий                                           |
| ---------------------------------------- | ------------------- | ----------------------------------------------------- |
| `~/.ssh/`                                | `700`               | Только владелец может читать/писать/выполнять         |
| `~/.ssh/id_rsa` или `id_ed25519`         | `600`               | Приватный ключ: только владелец может читать и писать |
| `~/.ssh/id_rsa.pub` или `id_ed25519.pub` | `644`               | Публичный ключ: может быть доступен всем              |
| `~/.ssh/authorized_keys`                 | `600`               | Список разрешённых публичных ключей (на сервере)      |
| `~/.ssh/known_hosts`                     | `644`               | Файл с фингерпринтами удалённых хостов                |
| `/etc/ssh/ssh_host_*` (на сервере)       | `600` / `644`       | Хост-ключи и их публичные части                       |
