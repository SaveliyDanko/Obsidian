#linux 

---
Команда `ln` создаёт ссылку на файл или каталог.
```bash
ln [опции] источник ссылка
```

###### ==Жёсткая ссылка (Hard link)==
```bash
ln файл.txt ссылка.txt
```
- Ссылка указывает не на имя файла, а на его содержимое на диске (inode)
- Оригинал и ссылка полностью равноправны — можно удалить оригинал, и ссылка всё равно будет работать
- Работает только внутри одного файлового раздела
- Нельзя создать на директории (по умолчанию)

###### ==Символическая ссылка (symlink)==
```bash
ln -s файл.txt ссылка.txt
```
- Ссылка указывает на путь к файлу
- Если оригинальный файл удалён — ссылка становится «битой»
- Можно создавать на каталоги
- Работает между разными файловыми системами
