#linux 

---
`apt` — это интерфейс командной строки для управления пакетами в дистрибутивах, основанных на Debian.

**1** `apt update` - обновляет список доступных пакетов из репозиториев

**2** `apt upgrade` - обновляет установленные пакеты до последних версий

**3** `apt install <пакет>` - устанавливает указанный пакет

**4** `apt remove <пакет>` - удаляет пакет, оставляя конфиги

**5** `apt purge <пакет>` - yдаляет пакет и все его конфиги

**6** `apt autoremove` - удаляет неиспользуемые зависимости

**7** `apt search <слово>` - ищет пакеты по имени или описанию

**8** `apt show <пакет>` - показывает подробную информацию о пакете

**9** `apt list --installed` - список установленных пакетов

**10** `apt clean` - удаляет загруженные `.deb` файлы из кэша

**11** `apt-mark showmanual` - показывает список установленных вручную пакетов
