`iptables` — это мощный инструмент и интерфейс командной строки для настройки сетевого брандмауэра в Linux. Он работает поверх **Netfilter**, встроенной подсистемы ядра Linux, которая фильтрует, перенаправляет и обрабатывает сетевые пакеты.

Iptables управляет таблицами, таблицы содержат цепочки, а цепочки содержат правила.

```zsh
iptables --help
```

###### Примеры
Разрешить входящие соединения на порт 22 (SSH):
```zsh
sudo iptables -A INPUT -p tcp --dport 22 -j ACCEPT
```

Разрешить весь исходящий трафик:
```zsh
sudo iptables -A OUTPUT -j ACCEPT
```
