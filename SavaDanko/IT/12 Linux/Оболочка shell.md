#linux 

---
Оболочка (shell) — это интерфейс между тобой и системой, который позволяет вводить команды, запускать программы, писать скрипты и автоматизировать работу.
###### Что такое оболочка (shell) в Linux?
- Это программа, которая запускается внутри терминала
- Обрабатывает введённые команды, переменные, пайпы, условия и циклы
- Примеры: `bash`, `zsh`, `fish`, `sh`

Просмотр оболочки:
```bash
echo $SHELL
```

###### `.bashrc`
`.bashrc` — это скрипт конфигурации, который выполняется при запуске интерактивной оболочки `bash`, например, когда ты открываешь новое окно терминала.

Он находится в домашней директории пользователя:
```bash
~/.bashrc
```
