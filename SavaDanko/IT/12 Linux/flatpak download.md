###### **Установка Flatpak**
Чтобы установить Flatpak на Ubuntu откройте терминал и выполните:
```bash
sudo apt install flatpak
```

**Установка плагина Flatpak для GNOME Software**
Плагин GNOME Software позволяет устанавливать приложения Flatpak без использования командной строки. Чтобы установить его, выполните:
```bash
sudo apt install gnome-software-plugin-flatpak
```
Примечание:  
Ubuntu распространяет GNOME Software как Snap-пакет в версиях 20.04 - 23.04 и заменяет его на App Center в версиях 23.10 и новее — ни один из них не поддерживает установку Flatpak-приложений.  
Установка плагина Flatpak также приведет к установке `.deb`-версии GNOME Software, что приведет к появлению двух приложений "Программы" одновременно на Ubuntu 20.04 - 23.04, и одного нового приложения "Программы" на Ubuntu 23.10 и новее.

**Добавление репозитория Flathub**
Flathub — это лучшее место для получения приложений Flatpak. Чтобы его подключить, выполните:
```bash
flatpak remote-add --if-not-exists flathub https://dl.flathub.org/repo/flathub.flatpakrepo
```

**Перезагрузка**
Чтобы завершить настройку, перезагрузите систему. После этого вы можете устанавливать приложения!
