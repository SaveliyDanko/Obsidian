Все переменные, начинающиеся со знака `$` (например, `$HOME`, `$PATH`, `$USER`, `$SHELL`), — это переменные окружения (environment variables), и их значения находятся в оперативной памяти, пока работает твоя оболочка (например, `bash` или `zsh`).

###### ==Посмотреть значение конкретной переменной:==
```bash
echo $HOME
echo $PATH
echo $USER
```

###### ==Показать все переменные окружения:==
```bash
env
```

###### ==Откуда берутся переменные окружения?==
Переменные загружаются из разных **конфигурационных файлов** в момент входа в терминал или оболочку:

###### ==Как задать переменную вручную?==
Временная переменная (на время текущей сессии):
```bash
export MY_VAR="Hello"
echo $MY_VAR
```

Чтобы задать **навсегда** — добавь в `~/.bashrc`, `~/.zshrc` или `~/.profile`:
```bash
export MY_VAR="Hello"
```

Затем:
```bash
source ~/.bashrc
```
