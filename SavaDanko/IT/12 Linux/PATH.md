#linux 

---
`PATH` — это системная переменная окружения, которая содержит список директорий, разделённых двоеточиями `:`
Каждая из этих директорий содержит исполняемые файлы. Когда ты вводишь команду (например, `ls`, `python`, `git`), система ищет её в этих папках, по порядку.

###### Пример значения переменной `PATH`
```bash
echo $PATH
```

Пример вывода:
```
/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin:/home/user/.local/bin
```

###### Зачем нужен `PATH`?
- Чтобы **не указывать полный путь** к командам каждый раз:
```bash
/usr/bin/ls → просто ls
~/.local/bin/kitty → просто kitty
```  


###### Как добавить папку в PATH?
Допустим, ты установил утилиту в `~/.local/bin` и хочешь запускать её из любой директории.

###### Временно (до перезагрузки терминала):
```bash
export PATH="$HOME/.local/bin:$PATH"
```

###### Постоянно (для всех будущих терминалов):
Добавь в конец файла `~/.bashrc`, `~/.zshrc` или `~/.profile`:
```bash
export PATH="$HOME/.local/bin:$PATH"
```

И перезагрузи конфигурацию:
```bash
source ~/.bashrc
```
