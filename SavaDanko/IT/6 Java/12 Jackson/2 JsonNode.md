`JsonNode` — это основной абстрактный класс в библиотеке Jackson, представляющий элемент JSON-дерева.
Он используется для tree-based модели работы с JSON: позволяет читать, обходить, изменять и анализировать JSON как структуру узлов.

###### **Назначение**
JsonNode нужен для:
- парсинга JSON в виде дерева (`ObjectMapper.readTree()`)
- удобного доступа к полям и массивам без создания POJO-классов
- динамической работы с неизвестной структурой JSON
- навигации по дереву и выборки значений по ключам и индексам
- преобразования JSON-узлов обратно в строки или объекты

###### **Основные типы узлов**
- `ObjectNode` — объект `{}`
- `ArrayNode` — массив `[]`
- `TextNode`, `IntNode`, `BooleanNode` и др. — примитивы

###### **Навигация**
- `get(String fieldName)` — получить узел по ключу (или `null`).
- `get(int index)` — получить элемент массива.
- `path(String name)` — как `get`, но никогда не возвращает null (возвращает MissingNode).
- `path(int index)` — безопасный доступ к массиву.
- `fields()` — получить итератор по полям объекта.
- `elements()` — итератор по элементам массива.

###### **Проверка типа**
- `isObject()`, `isArray()`, `isTextual()`, `isNumber()`, `isBoolean()`, etc.
- `isMissingNode()` — узел отсутствует.
- `isNull()` — JSON null.

###### **Извлечение значений**
- `asText()`, `asInt()`, `asLong()`, `asBoolean()` — получить значение с приведением.
- `textValue()`, `intValue()` и т.п. — получить точное значение без преобразований.

###### **Преобразования**
- `toString()` — получить JSON-строку.
- `deepCopy()` — глубокое копирование узла.
- `iterator()` — обход элементов (универсальный).
