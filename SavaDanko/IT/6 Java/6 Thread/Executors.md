Класс `java.util.concurrent.Executors` — это фабрика потоков, предоставляющая удобные методы для создания различных пулов потоков и планировщиков задач.

| Метод                                | Описание                                                       |
| ------------------------------------ | -------------------------------------------------------------- |
| `newFixedThreadPool(int n)`          | Создаёт пул с фиксированным количеством потоков.               |
| `newCachedThreadPool()`              | Пул, который динамически создаёт потоки по мере необходимости. |
| `newSingleThreadExecutor()`          | Один поток, последовательное выполнение задач.                 |
| `newScheduledThreadPool(int n)`      | Пул для планирования задач с задержкой или периодичностью.     |
| `newSingleThreadScheduledExecutor()` | Планировщик с одним потоком.                                   |
| `defaultThreadFactory()`             | Возвращает стандартную фабрику для создания потоков.           |
