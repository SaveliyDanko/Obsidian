Plugin Rainbow Brackets
Plugin Key Promoter X

###### ==Навигация по файлам, символам, классам==
`Ctrl + B` - Быстрый переход по коду
`Ctrl + N` — найти класс
`Ctrl + Shift + N` — найти файл
`Double Shift` — глобальный поиск по всему проекту (всё-в-одном)

###### ==Структура класса==
`Ctrl + Win + F12` — показать структуру текущего файла (все методы, поля и т.п.) в виде списка

###### ==Навигация по иерархии классов==
`Ctrl + H` — иерархия классов (показывает наследование вверх и вниз)

###### ==Refactoring Tools==
`Shift + F6` — переименование (переменной, метода, класса и т.д.)
`Ctrl + Alt + M` — извлечь код в метод
`Ctrl + Alt + V` — извлечь выражение в переменную    
`Ctrl + Alt + F` — извлечь в поле
`Ctrl + Alt + C` — извлечь в константу

###### ==Инспекции и исправления==
`Alt + Enter` — быстрые подсказки, исправления, добавление импорта, генерация кода и т.д.

###### ==Множественное выделение и редактирование==
`Alt + J` — выделить следующий такой же фрагмент
`Ctrl + G` — перейти к строке    
`Ctrl + W` — расширить выделение
`Ctrl + Shift + W` — сузить выделение
