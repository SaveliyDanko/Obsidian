A _class declaration_ specifies a class.
There are three kinds of class declarations: _normal class declarations_, _enum declarations_ ([§8.9](https://docs.oracle.com/javase/specs/jls/se24/html/jls-8.html#jls-8.9 "8.9. Enum Classes")), and _record declarations_ ([§8.10](https://docs.oracle.com/javase/specs/jls/se24/html/jls-8.html#jls-8.10 "8.10. Record Classes")).
```
ClassDeclaration:
	NormalClassDeclaration  
	EnumDeclaration
	RecordDeclaration

NormalClassDeclaration:
{ClassModifier} class  TypeIdentifier [TypeParameters] [ClassExtends] [ClassImplements] [ClassPermits] ClassBody
```
A class is also implicitly declared by a class instance creation expression ([§15.9.5](https://docs.oracle.com/javase/specs/jls/se24/html/jls-15.html#jls-15.9.5 "15.9.5. Anonymous Class Declarations")) and an enum constant that ends with a class body ([§8.9.1](https://docs.oracle.com/javase/specs/jls/se24/html/jls-8.html#jls-8.9.1 "8.9.1. Enum Constants")).
The _TypeIdentifier_ in a class declaration specifies the name of the class.
It is a compile-time error if a class has the same simple name as any of its enclosing classes or interfaces.
The scope and shadowing of a class declaration is specified in [§6.3](https://docs.oracle.com/javase/specs/jls/se24/html/jls-6.html#jls-6.3 "6.3. Scope of a Declaration") and [§6.4.1](https://docs.oracle.com/javase/specs/jls/se24/html/jls-6.html#jls-6.4.1 "6.4.1. Shadowing").
