#java 

---
Класс `Object` — это основа всего в Java. Он находится в пакете `java.lang`, и все классы в Java неявно наследуются от него, даже если ты этого явно не пишешь.
```java
public class Person {
    // неявно: extends Object
}
```
Класс `Object` предоставляет универсальные методы, которые доступны в любом классе.
