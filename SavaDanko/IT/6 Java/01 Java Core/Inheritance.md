---
**Наследование** — это механизм, при котором один класс может унаследовать поля и методы другого класса, расширяя или переопределяя их поведение.
```java
class Подкласс extends РодительскийКласс {
    // можно добавить или переопределить поведение
}
```


###### Переопределение метода (`@Override`)
```java
public class Dog extends Animal {
    @Override
    public void speak() {
        System.out.println("Гав-гав");
    }
}
```

###### Ключевое слово `super`
- Вызывает конструктор или метод суперкласса
```java
public class Dog extends Animal {
    @Override
    public void speak() {
        super.speak(); // вызов метода родителя
        System.out.println("И гав-гав");
    }
}
```
- Конструкторы не наследуются
- Подкласс всегда вызывает конструктор суперкласса (неявно или явно через `super()`)

| Ограничение                                        | Пример                          |
| -------------------------------------------------- | ------------------------------- |
| ❌ **Нет множественного наследования классов**      | `class C extends A, B` — ошибка |
| ✅ Есть **множественная имплементация интерфейсов** | `class C implements A, B` — ок  |
| ✅ Все классы неявно наследуют `Object`             | даже если ты этого не пишешь    |
