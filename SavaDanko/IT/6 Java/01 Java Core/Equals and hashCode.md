#java 

---
`equals(Object obj)` — метод, который определяет, равны ли два объекта по содержанию.
`hashCode()` — метод, возвращающий числовой код объекта, используемый, например, в `HashMap`, `HashSet` и других хеш-структурах.

##### Контракт `equals()` и `hashCode()`:
Если `a.equals(b) == true`, то обязательно `a.hashCode() == b.hashCode()`.  
НО обратное не всегда верно: одинаковые `hashCode` не гарантируют равенство по `equals()`.

##### Метод `equals()` должен быть:
- **рефлексивным**: `x.equals(x)` → `true`
- **симметричным**: `x.equals(y)` ⇔ `y.equals(x)`
- **транзитивным**: `x.equals(y) && y.equals(z)` ⇒ `x.equals(z)`
- **постоянным**: результат не должен меняться без изменения состояний
- **согласованным с `null`**: `x.equals(null)` → `false`

##### `hashCode()`:
- Должен возвращать одинаковое значение для равных объектов
- Должен быть как можно более равномерным, чтобы избежать коллизий в хеш-таблицах
