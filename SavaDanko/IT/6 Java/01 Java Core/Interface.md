#java 

---
**Интерфейс** — это набор абстрактных методов (и не только), которые класс обязан реализовать.
Интерфейс не содержит реализации, только что должен уметь объект.
```java
public interface Drawable {
    void draw(); // абстрактный метод
}
```

```java
public class Circle implements Drawable {
    @Override
    public void draw() {
        System.out.println("Рисуем круг");
    }
}
```

| Особенность                        | Есть?                                               |
| ---------------------------------- | --------------------------------------------------- |
| Поля                               | Да, но **только** `public static final` (константы) |
| Методы без тела (абстрактные)      | Да (по умолчанию)                                   |
| `default`-методы с реализацией     | ✅ С Java 8                                          |
| `static`-методы                    | ✅ С Java 8                                          |
| `private`-методы внутри интерфейса | ✅ С Java 9                                          |
| Конструкторы                       | ❌ Нет                                               |
