#java 

---
Ключевое слово `final` в Java — это ограничитель. Оно означает, что что-то нельзя изменить или переопределить. Оно может применяться к переменным, методам и классам, и в каждом случае действует по-своему.

###### 1. `final` для переменных
Означает, что переменная может быть присвоена только один раз.
final-ссылки на объекты запрещает менять ссылку, но не сам объект.

###### 2. `final` для методов
Означает, что метод нельзя переопределить в подклассах.

###### 3. `final` для классов
Означает, что класс нельзя наследовать.

###### 4. `final` и анонимные классы
Внутри анонимных классов можно использовать только `final` или `effective final` переменные
