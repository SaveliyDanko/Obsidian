#java 

---
Ключевое слово `abstract` в Java используется для создания абстрактных классов и методов, которые задают общую структуру, но не определяют точную реализацию. Это один из важнейших инструментов абстракции в объектно-ориентированном программировании.
- Абстрактный класс может содержать как реализованные, так и абстрактные методы.
- Абстрактный метод — это метод без тела (`{}`), только сигнатура.
- Нельзя создать объект абстрактного класса напрямую.
- Абстрактный метод должен быть реализован в подклассе, если только подкласс сам не абстрактный.
