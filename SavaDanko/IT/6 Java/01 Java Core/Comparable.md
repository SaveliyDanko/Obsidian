**`Comparable`** — это интерфейс, который позволяет объекту "сравнивать самого себя с другим объектом того же типа".

Ты реализуешь `Comparable`, когда:
- Хочешь отсортировать список объектов
- Используешь `TreeMap`, `TreeSet` или `PriorityQueue`, которые требуют "сортируемости";
##### Интерфейс `Comparable<T>`
```java
public interface Comparable<T> {
    int compareTo(T o);
}
```
- Возвращает:
    - `0`, если объекты равны
    - положительное число, если `this > o`
    - отрицательное число, если `this < o`

##### Ключевые правила:
1. Согласованность с `equals()`
2. Метод `compareTo()` должен быть рефлексивен, симметричен и транзитивен.
