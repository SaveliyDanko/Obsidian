---
**Полиморфизм** — это способность объектов разных классов реагировать на один и тот же вызов по-разному.
Другими словами: один интерфейс — много реализаций.

| Вид                     | Что это                                         |
| ----------------------- | ----------------------------------------------- |
| **Во время компиляции** | **Перегрузка методов** (method overloading)     |
| **Во время выполнения** | **Переопределение методов** (method overriding) |
