#java 

---
`StringBuilder` в Java — это класс из пакета `java.lang`, предназначенный для эффективной работы со строками, когда нужно часто изменять, добавлять или удалять символы.

В Java строки (`String`) неизменяемы — каждый раз, когда ты что-то добавляешь или меняешь, создаётся новая строка. Это неэффективно, особенно в циклах или при множестве операций.

`StringBuilder` решает эту проблему — он работает с изменяемой последовательностью символов, прямо в памяти.

Не потокобезопасен (в отличие от `StringBuffer`)
