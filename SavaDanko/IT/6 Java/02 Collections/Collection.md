#java 

---
**`Collection<E>`** — это **унифицированный интерфейс**, который представляет **группу объектов одного типа**, над которой можно выполнять базовые операции, такие как:
- Добавление элементов
- Удаление элементов
- Проверка наличия элементов
- Перебор всех элементов

```java
public interface Collection<E> extends Iterable<E> { ... }
```

| Метод                               | Что делает                                                  |
| ----------------------------------- | ----------------------------------------------------------- |
| `add(E e)`                          | Добавить элемент                                            |
| `addAll(Collection<? extends E> c)` | Добавить все элементы другой коллекции                      |
| `remove(Object o)`                  | Удалить элемент                                             |
| `removeAll(Collection<?> c)`        | Удалить все элементы, присутствующие в другой коллекции     |
| `retainAll(Collection<?> c)`        | Оставить только элементы, присутствующие в другой коллекции |
| `contains(Object o)`                | Проверить наличие элемента                                  |
| `containsAll(Collection<?> c)`      | Проверить наличие всех элементов                            |
| `size()`                            | Вернуть размер коллекции                                    |
| `clear()`                           | Очистить коллекцию                                          |
| `isEmpty()`                         | Проверить, пуста ли коллекция                               |
| `iterator()`                        | Вернуть `Iterator` для перебора                             |
| `toArray()`                         | Преобразовать в массив                                      |
