`LinkedList` — это **двусвязанный список** - структура, состоящая из узлов (Node), где каждый элемент ссылается на следующий и предыдущий.    

##### Методы (в дополнение к `List`)
`addFitst()` - вставка в начало
`addLast()` - вставка в конец
`removeFirst()` - удаление первого элемента
`removeLast()` - удаление последнего элемента
`peekFirst()` / `peekLast()` - получение первого/последнего без удаления
