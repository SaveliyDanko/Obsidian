###### Методы `ArrayList`:

| Метод               | Назначение                       |
| ------------------- | -------------------------------- |
| `add(E e)`          | Добавляет элемент в конец        |
| `add(int index, E)` | Вставляет элемент по индексу     |
| `get(int index)`    | Возвращает элемент по индексу    |
| `remove(int index)` | Удаляет элемент по индексу       |
| `remove(Object o)`  | Удаляет первый найденный элемент |
| `set(int index, E)` | Заменяет элемент                 |
| `size()`            | Возвращает размер списка         |
| `clear()`           | Очищает список                   |
| `contains(E e)`     | Проверяет наличие элемента       |
