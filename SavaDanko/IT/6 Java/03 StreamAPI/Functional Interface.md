**Функциональный интерфейс** — это интерфейс с ровно одним абстрактным методом.    

###### **Аннотация** `@FunctionalInterface`
```java
@FunctionalInterface
public interface MyAction {
    void execute();
}
```
- `@FunctionalInterface` не обязательна, но если ты её ставишь:
    - Компилятор проверит, что у интерфейса ровно один абстрактный метод.
    - Если случайно добавишь второй абстрактный метод — будет ошибка компиляции.

| Интерфейс        | Что делает                                               |
| ---------------- | -------------------------------------------------------- |
| `Runnable`       | Без аргументов, ничего не возвращает                     |
| `Callable<V>`    | Без аргументов, возвращает значение                      |
| `Supplier<T>`    | Поставщик значения (нет аргументов, возвращает значение) |
| `Consumer<T>`    | Принимает один аргумент, ничего не возвращает            |
| `Function<T, R>` | Принимает один аргумент и возвращает результат           |
| `Predicate<T>`   | Принимает аргумент и возвращает `boolean`                |
