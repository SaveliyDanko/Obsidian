**Route (маршрут) в Apache Camel** — это основная единица работы фреймворка, описывающая, как данные проходят путь “откуда → куда → что делать по дороге”.
- Route = цепочка шагов обработки данных
- Начинается с источника (`from(...)`)
- Заканчивается получателем (`to(...)`)
- Между ними можно добавлять обработку, фильтры, преобразования, логирование и т.д.
```java
from("file:input")
    .log("Processing ${file:name}")
    .to("file:output");
```
