**Camel Message** — это объект, который представляет данные, проходящие через маршрут (Route) в Apache Camel. Это контейнер, который несёт данные + метаданные.

###### **Из чего состоит Camel Message**
1. Body (тело сообщения)
Главные данные, которые обрабатываются маршрутом.  
Тип может быть любым: `String`, `File`, `JSON`, `byte[]`, `Map`, `Pojo`, etc.

2. Headers (заголовки сообщения)
Ключ-значение метаданных:
- имя файла
- Content-Type
- время получения
- значения из компонента (Kafka headers, HTTP headers)

3. MessageId + timestamp
	Служебные данные Camel:
	- уникальный ID сообщения
	- временная метка создания
