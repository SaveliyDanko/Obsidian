**Camel Exchange** — это основной объект обмена данными внутри маршрута (Route) в Apache Camel.
Он представляет собой контейнер, внутри которого лежит сообщение (Message) + контекст обработки.

###### **Что входит в Camel Exchange**
1. Exchange ID
Уникальный идентификатор обработки.

2. In Message
Входящее сообщение (body + headers), которое пришло от компонента-источника (например, File consumer, Kafka consumer, HTTP request).

3. Out Message
Сообщение, которое маршрут формирует как результат.
В современных версиях Camel (3.x, 4.x) это просто `exchange.getMessage()`.

4. Properties
Глобальные параметры на уровне Exchange, видимые всеми процессорами:
- путь файла
- время начала обработки
- внутренние параметры маршрута

5. Variables
Локальные переменные в Camel 4.x, похожие на properties, но “ближе” к маршруту.

6. Exception
Если в маршруте возникает ошибка — она хранится в Exchange.
