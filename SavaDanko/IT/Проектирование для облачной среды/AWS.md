###### **AWS**
- Публичное облако Amazon: платформа из сотен управляемых сервисов (вычисления, сети, данные, ИИ).
- Оплата за использование, быстрый старт инфраструктуры через консоль/CLI/API.
- Глобальная инфраструктура: регионы и зоны доступности для отказоустойчивости.

###### **S3 (Amazon Simple Storage Service)**
- Объектное хранилище для файлов/данных (бэкапы, статика, логи, ML-датасеты).
- “Бакеты” + версии/классы хранения, высокая долговечность, доступ по HTTP(S).

###### **EC2 (Elastic Compute Cloud)**
- Виртуальные машины “по требованию” (CPU/GPU/ARM, разные типы и размеры).
- Гибкая масштабируемость, автоскейлинг.
- Полный контроль над ОС/ПО, подходит для долгоживущих сервисов.

###### **SQS (Simple Queue Service)**
- Управляемая очередь сообщений для асинхронного взаимодействия сервисов.
