**DNS (Domain Name System)** — это система доменных имён, которая преобразует имена сайтов в IP-адреса.
- Главная функция: позволяет обращаться к сайтам по понятным именам (например, `google.com`), а не по числовым IP.

Как работает:
1. Пользователь вводит адрес сайта.
2. Компьютер запрашивает DNS-сервер.
3. Сервер возвращает соответствующий IP-адрес.
4. Браузер подключается к нужному серверу.
