**Протокол** — это набор правил и форматов, по которым устройства обмениваются данными в сети.

Он определяет:
- что можно передавать
- в каком виде    
- в каком порядке    
- и как понять, что сообщение принято правильно

###### **Пример в интернете**
Когда браузер обращается к сайту:
```
GET /index.html HTTP/1.1
Host: example.com
```
Он использует протокол HTTP — то есть строго определённую форму запроса.

Сервер обязан ответить в том же формате:
```
HTTP/1.1 200 OK
Content-Type: text/html
```
Если бы браузер отправил запрос в другом формате, сервер бы не понял его.
