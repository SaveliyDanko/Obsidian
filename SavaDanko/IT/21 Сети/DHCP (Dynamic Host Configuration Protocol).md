**DHCP (Dynamic Host Configuration Protocol)** — это протокол автоматической настройки сетевых устройств.
- Главная функция: автоматически выдаёт IP-адреса и другие сетевые параметры (маску, шлюз, DNS) устройствам в сети.

Как работает:
1. Устройство отправляет запрос на IP (DHCP Discover).
2. Сервер предлагает адрес (DHCP Offer).
3. Клиент принимает (DHCP Request).
4. Сервер подтверждает (DHCP Acknowledge).

Преимущества:
- Не нужно настраивать IP вручную.    
- Исключаются конфликты адресов.
- Упрощается администрирование сети.
