**NAT (Network Address Translation)** — это технология преобразования IP-адресов при передаче данных между сетями.
- Главная функция: позволяет нескольким устройствам с частными IP выходить в интернет через один внешний IP-адрес.

Как работает:
- Устройство отправляет пакет во внешнюю сеть.    
- Роутер подменяет внутренний IP на свой внешний.
- При ответе — восстанавливает исходный адрес и пересылает обратно.
