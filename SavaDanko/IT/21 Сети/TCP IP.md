Если модель **OSI** — это теоретическая 7-уровневая модель, то **TCP/IP** — это практическая, реально используемая в интернете модель, на которой действительно работает всё сетевое взаимодействие.

| Уровень TCP/IP                                     | Аналог в OSI   | Что делает                                                                  |
| -------------------------------------------------- | -------------- | --------------------------------------------------------------------------- |
| Прикладной (Application)                           | 7–5 уровни OSI | Всё, что касается приложений и протоколов вроде HTTP, FTP, DNS, SMTP и т.д. |
| Транспортный (Transport)                           | 4 уровень OSI  | Обеспечивает доставку данных между программами — TCP, UDP                   |
| Интернет (Internet)                                | 3 уровень OSI  | Отвечает за маршрутизацию и адресацию пакетов — IP, ICMP, ARP               |
| Канальный / Сетевой доступ (Network Access / Link) | 2–1 уровни OSI | Физическая передача данных — Ethernet, Wi-Fi, драйверы, сигналы             |

###### **Канальный уровень (Network Access Layer)**
Передаёт данные между устройствами на одной физической сети.  
Использует MAC-адреса, Ethernet-кадры, Wi-Fi, Bluetooth и т.д.

###### **Интернет-уровень (Internet Layer)**
Главный протокол: IP (Internet Protocol).  
Он решает, куда именно доставить пакет — по IP-адресу.

###### **Транспортный уровень (Transport Layer)**
Передаёт данные от приложения к приложению, обеспечивая надёжность и управление потоком.

| Протокол                            | Особенности                                          |
| ----------------------------------- | ---------------------------------------------------- |
| TCP (Transmission Control Protocol) | Надёжный, с подтверждением доставки, контроль потока |
| UDP (User Datagram Protocol)        | Быстрый, без подтверждений, возможна потеря пакетов  |

###### **Прикладной уровень (Application Layer)**
Даёт приложениям возможность использовать сеть — здесь работают программы, знакомые пользователю.
