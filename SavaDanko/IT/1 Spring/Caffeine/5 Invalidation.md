**Invalidation** в Caffeine — это ручное удаление записей из кэша.

В отличие от Eviction (автоматическое удаление), Invalidation выполняется по команде разработчика.

###### **Основные методы:**
```java
cache.invalidate(key);        // удалить один ключ
cache.invalidateAll(keys);    // удалить несколько ключей
cache.invalidateAll();        // очистить весь кэш
```

###### **Когда используется:**
- когда данные изменились в базе и их нужно освежить
- когда кэш нужно очистить перед перезагрузкой
- когда нужно принудительно сбросить старые значения
