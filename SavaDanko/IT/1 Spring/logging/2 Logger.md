Класс `Logger` из библиотеки **SLF4J** используется для записи логов на различных уровнях (INFO, DEBUG, ERROR и т.д.).  
Это **интерфейс**, который определяет основные методы для логирования.

| Метод                                 | Описание                                                  |
| ------------------------------------- | --------------------------------------------------------- |
| `info(String msg)`                    | Логирует сообщение на уровне **INFO**.                    |
| `debug(String msg)`                   | Логирует сообщение на уровне **DEBUG**.                   |
| `warn(String msg)`                    | Логирует сообщение на уровне **WARN**.                    |
| `error(String msg)`                   | Логирует сообщение на уровне **ERROR**.                   |
| `trace(String msg)`                   | Логирует сообщение на уровне **TRACE** (подробный вывод). |
| `isInfoEnabled()`                     | Проверяет, включено ли логирование на уровне **INFO**.    |
| `isDebugEnabled()`                    | Проверяет, включено ли логирование на уровне **DEBUG**.   |
| `error(String msg, Throwable t)`      | Логирует сообщение об ошибке с исключением.               |
| `info(String format, Object... args)` | Логирует форматированное сообщение.                       |
