Класс `LoggerContext` из библиотеки **Logback** представляет собой контейнер для логгеров и конфигураций. Он управляет иерархией логгеров и хранит общие настройки логирования.  
Это основной класс при работе с конфигурацией логгирования.

| Метод                                   | Описание                                                      |
| --------------------------------------- | ------------------------------------------------------------- |
| `getLogger(String name)`                | Возвращает логгер с указанным именем.                         |
| `getLogger(Class<?> clazz)`             | Возвращает логгер, связанный с указанным классом.             |
| `stop()`                                | Останавливает контекст логгирования.                          |
| `start()`                               | Запускает контекст логгирования.                              |
| `reset()`                               | Сбрасывает все настройки логгера к начальным.                 |
| `getStatusManager()`                    | Получает статусный менеджер (для логирования ошибок Logback). |
| `getProperty(String key)`               | Возвращает значение свойства по ключу.                        |
| `putProperty(String key, String value)` | Устанавливает пользовательское свойство.                      |
| `getLoggerList()`                       | Возвращает список всех зарегистрированных логгеров.           |
