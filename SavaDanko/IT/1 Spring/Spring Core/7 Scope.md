**Bean Scope**  — это правило, определяющее:
- сколько экземпляров объекта будет создано
- где и как долго он будет жить в контексте приложения

| Scope         | Описание                                                  |
| ------------- | --------------------------------------------------------- |
| `singleton`   | Один экземпляр на весь Spring-контейнер (по умолчанию)    |
| `prototype`   | Новый экземпляр каждый раз при запросе                    |
| `request`     | Один бин на каждый HTTP-запрос (только в web-приложениях) |
| `session`     | Один бин на одну HTTP-сессию                              |
| `application` | Один бин на весь `ServletContext` (всё веб-приложение)    |
| `websocket`   | Один бин на одно WebSocket-соединение                     |

```java
@Component
@Scope("singleton") // можно не писать, это поведение по умолчанию
public class Engine {}
```
