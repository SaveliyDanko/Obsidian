###### **Как указать, какой бин внедрять, если их несколько?**
Используй `@Qualifier`:
```java
@Component
public class PetrolEngine implements Engine {}

@Component
public class ElectricEngine implements Engine {}

@Component
public class Car {
    @Autowired
    public Car(@Qualifier("electricEngine") Engine engine) {
        this.engine = engine;
    }
}
```

Или пометь один из бинов `@Primary`, чтобы его использовать по умолчанию:
```java
@Primary
@Component
public class ElectricEngine implements Engine {}
```
