`Session` - обеспечивает интерфейс для выполнения операций с базой данных (CRUD, транзакции, запросы).
- CRUD операции: Создание, чтение, обновление и удаление данных.
- Управление транзакциями: Начало, коммит и откат транзакций.
- Запросы к базе данных: HQL-запросы, SQL-запросы, именованные запросы.
- Кэширование данных: Поддержка кэша первого уровня (на уровне сессии).
- Жизненный цикл: Сессия может быть в открытом или закрытом состоянии.
```java
Session session = sessionFactory.getCurrentSession();
session.beginTransaction();  
session.persist(person);  
session.getTransaction().commit();
```
