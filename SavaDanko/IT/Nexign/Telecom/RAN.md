**RAN (радиосетевая подсистема, Radio Access Network)** — это часть телекоммуникационной сети, которая соединяет абонентское устройство (телефон, модем, IoT-устройство) с ядром сети (Core Network) через радиоинтерфейс.  
Именно RAN обеспечивает доступ к мобильной сети через базовые станции и управляет радиоресурсами.

###### **Основные элементы RAN**
1. User Equipment (UE)
    - телефоны, модемы, планшеты, IoT-девайсы.
        
2. Базовые станции
    - В 2G → BTS (Base Transceiver Station)
    - В 3G → NodeB
    - В 4G → eNodeB (Evolved NodeB)
    - В 5G → gNodeB (Next Generation NodeB)
       
3. Transport Network
    - каналы, связывающие базовые станции с ядром сети (по оптике, микроволновым каналам и т.д.).

###### **Основные функции RAN**
- Управление радиоинтерфейсом (сигнализация, выбор частот, модуляция, кодирование).
- Управление подключением (handover между сотами, регистрация абонента).
- Распределение ресурсов (назначение полосы пропускания).
- Мультиплексирование трафика (одновременно передавать голоса, данные, видео).
- Безопасность (шифрование на уровне радиоинтерфейса).
