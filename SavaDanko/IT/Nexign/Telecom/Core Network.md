**Core Network (ядро сети)** — это “мозг” телекоммуникационной сети.  
Если RAN (Radio Access Network) обеспечивает радиодоступ, то Core Network отвечает за всё, что происходит дальше: маршрутизацию трафика, управление абонентами, услуги, биллинг и безопасность.

Иными словами, Core соединяет RAN с внешними сетями (Интернет, PSTN, корпоративные сети) и управляет всей логикой обслуживания.

###### **Основные функции Core Network**
1. Регистрация и аутентификация абонентов
    - проверка SIM/eSIM, ключей, разрешение доступа.
        
2. Управление сессиями и услугами
    - настройка соединений (голос, данные, видео, IoT).
    - контроль качества обслуживания (QoS).
        
3. Маршрутизация и коммутирование
    - выбор оптимального пути трафика.
    - интеграция с Интернетом, телефонной сетью, корпоративными VPN.
        
4. Управление мобильностью
    - отслеживание местоположения устройства. 
    - handover между сотами и даже сетями (например, 4G → 5G).
        
5. Тарификация и биллинг
    - генерация CDR (Call Detail Records). 
    - взаимодействие с биллинговыми системами (prepaid/postpaid).
        
6. Безопасность
    - шифрование трафика. 
    - защита от несанкционированного доступа.
