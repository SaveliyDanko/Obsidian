В Gradle **plugins** — это расширения, которые добавляют дополнительную функциональность в сборочный процесс.

#### Как подключаются плагины
```groovy
plugins {
    id 'java'                      // для Java-проекта
    id 'application'              // для запуска приложения
    id 'org.springframework.boot' version '3.2.5' // пример стороннего плагина
}
```
